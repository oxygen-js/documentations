# Знакомство со средством запуска тестов Node.js

В этой серии материалов мы узнаем, как использовать средство запуска тестов Node.js для тестирования нашего кода.

## Что такое средство запуска тестов?

Средство запуска тестов — это инструмент, который позволяет запускать тесты для вашего кода. Оно выполнит ваши тесты и предоставит вам обратную связь о том, прошли они или провалились. Оно также может предоставить дополнительную информацию, например покрытие кода.

Для Node.js доступно множество средств запуска тестов, но мы сосредоточимся на встроенном средстве запуска тестов, которое поставляется с Node.js. Оно замечательно тем, что для его использования не нужно устанавливать никаких дополнительных зависимостей.

## Зачем тестировать свой код?

Тестирование кода важно, потому что оно позволяет убедиться, что ваш код работает так, как ожидается. Это помогает выявлять ошибки на ранних этапах процесса разработки и обеспечивает надёжность и удобство сопровождения вашего кода.

## Запуск тестов с помощью Node.js

Чтобы запускать тесты с помощью Node.js, рекомендуем прочитать [документацию по средству запуска тестов](https://nodejs.org/docs/latest/api/test.html#test-runner).
