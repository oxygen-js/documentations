# Эмиттер событий Node.js

Если вы работали с JavaScript в браузере, вы знаете, насколько значительная часть взаимодействия пользователя обрабатывается через события: клики мышью, нажатия клавиш на клавиатуре, реакция на движения мыши и так далее.

Со стороны бэкенда Node.js предлагает нам возможность построить похожую систему с помощью [модуля `events`](https://nodejs.org/api/events.html).

В частности, этот модуль предоставляет класс `EventEmitter`, который мы будем использовать для обработки наших событий.

Вы инициализируете его так:
```js
const EventEmitter = require('node:events');

const eventEmitter = new EventEmitter();
```

Этот объект предоставляет, среди многих других, методы `on` и `emit`.

  * `emit` используется для запуска события
  * `on` используется для добавления функции-колбэка, которая будет выполнена при запуске события

Например, давайте создадим событие `start` и, чтобы привести образец, отреагируем на него, просто выведя сообщение в консоль:
```js
eventEmitter.on('start', () => {
  console.log('started');
});
```

Когда мы выполняем
```
eventEmitter.emit('start');
```

функция-обработчик события запускается, и мы получаем вывод в консоль.

Вы можете передавать аргументы обработчику события, передавая их как дополнительные аргументы в `emit()`:
```js
eventEmitter.on('start', number => {
  console.log(`started ${number}`);
});

eventEmitter.emit('start', 23);
```

Несколько аргументов:
```js
eventEmitter.on('start', (start, end) => {
  console.log(`started from ${start} to ${end}`);
});

eventEmitter.emit('start', 1, 100);
```

Объект EventEmitter также предоставляет несколько других методов для взаимодействия с событиями, например:

  * `once()`: добавить одноразовый слушатель
  * `removeListener()` / `off()`: удалить слушатель события у события
  * `removeAllListeners()`: удалить все слушатели для события

Подробнее об этих методах вы можете прочитать в [официальной документации](https://nodejs.org/api/events.html).
