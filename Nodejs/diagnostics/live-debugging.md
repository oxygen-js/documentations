# Отладка вживую (Live Debugging)

В этом документе вы можете узнать, как отлаживать процесс Node.js вживую.

## Моё приложение ведёт себя не так, как ожидалось

### Симптомы

Пользователь может заметить, что приложение не выдаёт ожидаемый результат для определённых входных данных, например, HTTP-сервер возвращает JSON-ответ, в котором некоторые поля пусты. В процессе может пойти не так множество вещей, но в этом сценарии мы в основном сосредоточены на логике приложения и её корректности.

### Отладка

В этом сценарии пользователь хотел бы понять, какой путь выполнения кода проходит наше приложение при определённом триггере, например при входящем HTTP-запросе. Он также может захотеть пройти по коду пошагово и контролировать выполнение, а также проверить, какие значения хранят переменные в памяти.

  * [Использование Inspector](/learn/diagnostics/live-debugging/using-inspector)
