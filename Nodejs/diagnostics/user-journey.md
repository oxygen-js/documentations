# Пользовательский сценарий

Эти руководства по диагностике были созданы [рабочей группой по диагностике (Diagnostics Working Group)](https://github.com/nodejs/diagnostics) и [командой веб-сайта Node.js (Node.js Website Team)](https://github.com/nodejs/nodejs.org) с целью предоставить руководство при диагностике проблемы в приложении пользователя.

Проект документации организован по принципу пользовательских сценариев. Эти сценарии представляют собой согласованный набор пошаговых процедур, которым пользователь может следовать, чтобы найти первопричину своих проблем.
