# Шпаргалка по Markdown

## Обзор

Эта шпаргалка по Markdown даёт краткий обзор всех элементов синтаксиса Markdown. Она не может охватить все частные случаи, поэтому, если вам нужно больше сведений о любом из этих элементов, обратитесь к справочным руководствам по [базовому синтаксису](<03. Basic Syntax.md>) и [расширенному синтаксису](<04. Extended Syntax.md>).

## Расширенный синтаксис

Эти элементы расширяют базовый синтаксис, добавляя дополнительные возможности. Не все приложения Markdown поддерживают эти элементы.

<table class="table table-bordered">
  <thead class="thead-light">
    <tr>
      <th>Элемент</th>
      <th>Синтаксис Markdown</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><a href="04.%20Extended%20Syntax.md#таблицы">Таблица</a></td>
      <td><code>
          | Syntax      | Description |<br>
          | ----------- | ----------- |<br>
          | Header      | Title       |<br>
          | Paragraph   | Text        |
      </code></td>
    </tr>
    <tr>
      <td><a href="04.%20Extended%20Syntax.md#огороженные-блоки-кода">Огороженный блок кода</a></td>
      <td><code>```<br>
      {<br>
      &nbsp;&nbsp;"firstName": "John",<br>
      &nbsp;&nbsp;"lastName": "Smith",<br>
      &nbsp;&nbsp;"age": 25<br>
      }<br>
      ```
      </code></td>
    </tr>
    <tr>
      <td><a href="04.%20Extended%20Syntax.md#сноски">Сноска</a></td>
      <td><code>
        Here's a sentence with a footnote. [^1]<br><br>
        [^1]: This is the footnote.
      </code></td>
    </tr>
    <tr>
      <td><a href="04.%20Extended%20Syntax.md#идентификаторы-заголовков">Идентификатор заголовка</a></td>

    </tr>
    <tr>
      <td><a href="04.%20Extended%20Syntax.md#списки-определений">Список определений</a></td>
      <td><code>
        term<br>
        : definition
      </code></td>
    </tr>
    <tr>
      <td><a href="04.%20Extended%20Syntax.md#зачёркивание">Зачёркивание</a></td>
      <td><code>~~The world is flat.~~</code></td>
    </tr>
    <tr>
      <td><a href="04.%20Extended%20Syntax.md#списки-задач">Список задач</a></td>
      <td><code>
        - [x] Write the press release<br>
        - [ ] Update the website<br>
        - [ ] Contact the media
      </code></td>
    </tr>
    <tr>
      <td><a href="04.%20Extended%20Syntax.md#эмодзи">Эмодзи</a><br>(см. также <a href="04.%20Extended%20Syntax.md#копирование-и-вставка-эмодзи">Копирование и вставка эмодзи</a>)</td>
