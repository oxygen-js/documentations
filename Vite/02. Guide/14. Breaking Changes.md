# Обратно несовместимые изменения

Список обратно несовместимых изменений в Vite, включая устаревание, удаление и изменение API. Большинство приведённых ниже изменений можно включить по желанию с помощью [опции `future`](https://vite.dev/config/shared-options.html#future) в вашей конфигурации Vite.

## Запланированные

Эти изменения запланированы к следующей мажорной версии Vite. Предупреждения об устаревании или использовании по возможности подскажут вам, что нужно сделать, и мы связываемся с авторами фреймворков, плагинов и пользователями, чтобы применить эти изменения.

- [`this.environment` в хуках](https://vite.dev/changes/this-environment-in-hooks)
- [Хук плагина `hotUpdate` для HMR](https://vite.dev/changes/hotupdate-hook)
- [SSR с использованием API `ModuleRunner`](https://vite.dev/changes/ssr-using-modulerunner)

## Рассматриваемые

Эти изменения находятся на рассмотрении и часто являются экспериментальными API, которые призваны улучшить текущие способы использования. Поскольку здесь перечислены не все изменения, полный список смотрите в разделе [метки Experimental в GitHub Discussions Vite](https://github.com/vitejs/vite/discussions/categories/feedback?discussions_q=label%3Aexperimental+category%3AFeedback).

Мы пока не рекомендуем переходить на эти API. Они включены в Vite, чтобы помочь нам собрать обратную связь. Пожалуйста, ознакомьтесь с этими предложениями и расскажите нам, как они работают в вашем случае, в соответствующих обсуждениях на GitHub.

- [Переход на API для конкретных окружений](https://vite.dev/changes/per-environment-apis)
- [Общие плагины во время сборки](https://vite.dev/changes/shared-plugins-during-build)
