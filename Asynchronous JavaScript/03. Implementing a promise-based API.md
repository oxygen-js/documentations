# Как реализовать API на основе промисов

В прошлой статье мы обсудили, как использовать API, возвращающие промисы. В этой статье мы рассмотрим другую сторону — как _реализовать_ API, возвращающие промисы. Это гораздо менее распространённая задача, чем использование API на основе промисов, но о ней всё равно стоит знать.

| | |
|---|---|
| **Предварительные требования:** | Уверенное понимание [основ JavaScript](https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/Scripting) и асинхронных концепций, рассмотренных в предыдущих уроках этого модуля. |
| **Результаты обучения:** | Понять, как реализовывать API на основе промисов. |

Как правило, реализуя API на основе промисов, вы будете оборачивать асинхронную операцию, которая может использовать события, обычные колбэки или модель передачи сообщений. Вы устроите так, чтобы объект `Promise` корректно обрабатывал успех или неудачу этой операции.

## Реализация API alarm()

В этом примере мы реализуем API будильника на основе промисов, который назовём `alarm()`. В качестве аргументов он будет принимать имя человека, которого нужно разбудить, и задержку в миллисекундах, которую нужно выждать перед пробуждением. По истечении задержки функция отправит сообщение «Wake up!» («Просыпайся!»), включающее имя человека, которого нужно разбудить.

### Оборачивание setTimeout()

Для реализации нашей функции `alarm()` мы воспользуемся API [`setTimeout()`](https://developer.mozilla.org/en-US/docs/Web/API/Window/setTimeout). API `setTimeout()` принимает в качестве аргументов функцию-колбэк и задержку, заданную в миллисекундах. Когда вызывается `setTimeout()`, он запускает таймер, установленный на заданную задержку, и по истечении времени вызывает переданную функцию.

В приведённом ниже примере мы вызываем `setTimeout()` с функцией-колбэком и задержкой в 1000 миллисекунд:

```html
<button id="set-alarm">Set alarm</button>
<div id="output"></div>
```

```css hidden
div {
  margin: 0.5rem 0;
}
```

```js
const output = document.querySelector("#output");
const button = document.querySelector("#set-alarm");

function setAlarm() {
  setTimeout(() => {
    output.textContent = "Wake up!";
  }, 1000);
}

button.addEventListener("click", setAlarm);
```

### Конструктор Promise()

Наша функция `alarm()` будет возвращать `Promise`, который выполняется по истечении таймера. Он будет передавать сообщение «Wake up!» в обработчик `then()` и отклонять промис, если вызывающий код передаст отрицательное значение задержки.

Ключевой компонент здесь — конструктор [`Promise()`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Promise/Promise). Конструктор `Promise()` принимает в качестве аргумента одну функцию. Мы будем называть эту функцию **исполнителем (executor)**. Создавая новый промис, вы предоставляете реализацию исполнителя.

Сама эта функция-исполнитель принимает два аргумента, которые тоже являются функциями и которые по соглашению называют `resolve` и `reject`. В своей реализации исполнителя вы вызываете нижележащую асинхронную функцию. Если асинхронная функция завершается успехом, вы вызываете `resolve`, а если неудачей — вызываете `reject`. Если функция-исполнитель выбрасывает ошибку, `reject` вызывается автоматически. В `resolve` и `reject` можно передать единственный параметр любого типа.

Итак, мы можем реализовать `alarm()` так:

```js
function alarm(person, delay) {
  return new Promise((resolve, reject) => {
    if (delay < 0) {
      reject(new Error("Alarm delay must not be negative"));
      return;
    }
    setTimeout(() => {
      resolve(`Wake up, ${person}!`);
    }, delay);
  });
}
```

Эта функция создаёт и возвращает новый `Promise`. Внутри исполнителя промиса мы:

- проверяем, что `delay` не отрицательна, и, если это не так, вызываем `reject`, передавая пользовательскую ошибку;

- вызываем `setTimeout()`, передавая колбэк и `delay`. Колбэк будет вызван по истечении таймера, и внутри него мы вызываем `resolve`, передавая наше сообщение «Wake up!».

## Использование API alarm()

Эта часть должна быть вам вполне знакома по прошлой статье. Мы можем вызвать `alarm()`, а на возвращённом промисе вызвать `then()` и `catch()`, чтобы задать обработчики выполнения и отклонения промиса.

```html hidden
<div>
  <label for="name">Name:</label>
  <input type="text" id="name" name="name" size="4" value="Matilda" />
</div>

<div>
  <label for="delay">Delay:</label>
  <input type="text" id="delay" name="delay" size="4" value="1000" />
</div>

<button id="set-alarm">Set alarm</button>
<div id="output"></div>
```

```css hidden
button {
  display: block;
}

div,
button {
  margin: 0.5rem 0;
}
```

```js
const name = document.querySelector("#name");
const delay = document.querySelector("#delay");
const button = document.querySelector("#set-alarm");
const output = document.querySelector("#output");

function alarm(person, delay) {
  return new Promise((resolve, reject) => {
    if (delay < 0) {
      reject(new Error("Alarm delay must not be negative"));
      return;
    }
    setTimeout(() => {
      resolve(`Wake up, ${person}!`);
    }, delay);
  });
}

button.addEventListener("click", () => {
  alarm(name.value, delay.value)
    .then((message) => (output.textContent = message))
    .catch((error) => (output.textContent = `Couldn't set alarm: ${error}`));
});
```

## Использование async и await с API alarm()

Поскольку `alarm()` возвращает `Promise`, мы можем делать с ним всё то же, что и с любым другим промисом: объединять в цепочки, использовать `Promise.all()` и `async` / `await`:

```html hidden
<div>
  <label for="name">Name:</label>
  <input type="text" id="name" name="name" size="4" value="Matilda" />
</div>

<div>
  <label for="delay">Delay:</label>
  <input type="text" id="delay" name="delay" size="4" value="1000" />
</div>

<button id="set-alarm">Set alarm</button>
<div id="output"></div>
```

```css hidden
button {
  display: block;
}

div,
button {
  margin: 0.5rem 0;
}
```

```js
const name = document.querySelector("#name");
const delay = document.querySelector("#delay");
const button = document.querySelector("#set-alarm");
const output = document.querySelector("#output");

function alarm(person, delay) {
  return new Promise((resolve, reject) => {
    if (delay < 0) {
      reject(new Error("Alarm delay must not be negative"));
      return;
    }
    setTimeout(() => {
      resolve(`Wake up, ${person}!`);
    }, delay);
  });
}

button.addEventListener("click", async () => {
  try {
    const message = await alarm(name.value, delay.value);
    output.textContent = message;
  } catch (error) {
    output.textContent = `Couldn't set alarm: ${error}`;
  }
});
```
