# Репозиторий

`Repository` работает так же, как `EntityManager`, но его операции ограничены одной конкретной сущностью.
Получить репозиторий можно через EntityManager.

Пример:

```typescript
import { User } from "./entity/User"

const userRepository = dataSource.getRepository(User)
const user = await userRepository.findOneBy({
    id: 1,
})
user.name = "Umed"
await userRepository.save(user)
```

Есть 3 типа репозиториев:

- `Repository` — обычный репозиторий для любой сущности.
- `TreeRepository` — репозиторий, расширение `Repository`, используемое для древовидных сущностей
  (то есть сущностей, помеченных декоратором `@Tree`).
  Содержит специальные методы для работы с древовидными структурами.
- `MongoRepository` — репозиторий со специальными функциями, применимыми только к MongoDB.
