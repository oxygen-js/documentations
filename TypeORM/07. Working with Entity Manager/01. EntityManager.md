# EntityManager

С помощью `EntityManager` вы можете управлять (вставлять, обновлять, удалять, загружать и т. д.) любой сущностью.
EntityManager — это как бы набор всех репозиториев сущностей, собранный в одном месте.

Получить доступ к entity manager можно через DataSource.
Пример использования:

```typescript
import { DataSource } from "typeorm"
import { User } from "./entity/User"

const myDataSource = new DataSource(/*...*/)
const user = await myDataSource.manager.findOneBy(User, {
    id: 1,
})
user.name = "Umed"
await myDataSource.manager.save(user)
```
