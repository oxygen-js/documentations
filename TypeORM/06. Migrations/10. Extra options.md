# Дополнительные опции

## Метка времени

Если нужно задать метку времени для имени миграции, используйте `-t` (сокращение от `--timestamp`) и передайте метку времени (должна быть неотрицательным числом)

```shell
typeorm -t <specific-timestamp> migration:{create|generate}
```

Получить метку времени можно так:

```js
Date.now()
/* ИЛИ */ new Date().getTime()
```
