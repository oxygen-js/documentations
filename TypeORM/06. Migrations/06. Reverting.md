# Откат миграций

Если по какой-то причине вы хотите откатить изменения, выполните:

```shell
typeorm migration:revert -- -d path-to-datasource-config
```

Эта команда выполнит `down` в последней применённой миграции.

Если нужно откатить несколько миграций, вызовите эту команду соответствующее число раз.
