# Статус

Чтобы показать все миграции и то, выполнены они или нет, используйте следующую команду:

```shell
typeorm migration:show  -- -d path-to-datasource-config
```

[X] = миграция выполнена

[ ] = миграция ожидает выполнения / не применена
