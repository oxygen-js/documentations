# Удаление с помощью Query Builder

## `Delete`

Запросы `DELETE` можно создавать через `QueryBuilder`.
Примеры:

```typescript
await myDataSource
    .createQueryBuilder()
    .delete()
    .from(User)
    .where("id = :id", { id: 1 })
    .execute()
```

Это самый эффективный по производительности способ удалить сущности из базы данных.

## `Soft-Delete`

Применение мягкого удаления в QueryBuilder

```typescript
await dataSource.getRepository(Entity).createQueryBuilder().softDelete()
```

Примеры:

```typescript
await myDataSource
    .getRepository(User)
    .createQueryBuilder()
    .softDelete()
    .where("id = :id", { id: 1 })
    .execute()
```

## `Restore-Soft-Delete`

Кроме того, мягко удалённые строки можно восстановить методом `restore()`:

```typescript
await dataSource.getRepository(Entity).createQueryBuilder().restore()
```

Примеры:

```typescript
await myDataSource
    .getRepository(User)
    .createQueryBuilder()
    .restore()
    .where("id = :id", { id: 1 })
    .execute()
```
