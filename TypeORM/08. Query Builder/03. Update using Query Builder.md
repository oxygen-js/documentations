# Обновление через Query Builder

Запросы `UPDATE` можно создавать с помощью `QueryBuilder`.
Примеры:

```typescript
await dataSource
    .createQueryBuilder()
    .update(User)
    .set({ firstName: "Timber", lastName: "Saw" })
    .where("id = :id", { id: 1 })
    .execute()
```

Это наиболее эффективный по производительности способ обновлять сущности в базе данных.

## Поддержка «сырого» SQL

В некоторых случаях, когда нужно выполнить SQL-запросы, значение задают в виде функции:

```typescript
await dataSource
    .createQueryBuilder()
    .update(User)
    .set({
        firstName: "Timber",
        lastName: "Saw",
        age: () => "age + 1",
    })
    .where("id = :id", { id: 1 })
    .execute()
```

> **Предупреждение** При использовании «сырого» SQL следите за тем, чтобы значения были должным образом экранированы — это защитит от SQL-инъекций.
