# Использование валидации

Для валидации используйте [class-validator](https://github.com/pleerock/class-validator).
Пример использования class-validator вместе с TypeORM:

```typescript
import { Entity, PrimaryGeneratedColumn, Column } from "typeorm"
import {
    Contains,
    IsInt,
    Length,
    IsEmail,
    IsFQDN,
    IsDate,
    Min,
    Max,
} from "class-validator"

@Entity()
export class Post {
    @PrimaryGeneratedColumn()
    id: number

    @Column()
    @Length(10, 20)
    title: string

    @Column()
    @Contains("hello")
    text: string

    @Column()
    @IsInt()
    @Min(0)
    @Max(10)
    rating: number

    @Column()
    @IsEmail()
    email: string

    @Column()
    @IsFQDN()
    site: string

    @Column()
    @IsDate()
    createDate: Date
}
```

Валидация:

```typescript
import { validate } from "class-validator"

let post = new Post()
post.title = "Hello" // не должно пройти проверку
post.text = "this is a great post about hell world" // не должно пройти проверку
post.rating = 11 // не должно пройти проверку
post.email = "google.com" // не должно пройти проверку
post.site = "googlecom" // не должно пройти проверку

const errors = await validate(post)
if (errors.length > 0) {
    throw new Error(`Validation failed!`)
} else {
    await dataSource.manager.save(post)
}
```
