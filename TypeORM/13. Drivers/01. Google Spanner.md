# Google Spanner

## Установка

```shell
npm install @google-cloud/spanner
```

## Опции источника данных

Общие опции источника данных описаны в разделе [Опции источника данных](<../02. Data Source/02. Data Source Options.md>).

Передайте учётные данные для аутентификации в код приложения,
задав переменную окружения `GOOGLE_APPLICATION_CREDENTIALS`:

```shell
# Linux/macOS
export GOOGLE_APPLICATION_CREDENTIALS="KEY_PATH"

# Windows
set GOOGLE_APPLICATION_CREDENTIALS=KEY_PATH

# Замените KEY_PATH на путь к JSON-файлу с ключом вашего сервисного аккаунта.
```

Чтобы использовать Spanner с эмулятором, задайте переменную окружения `SPANNER_EMULATOR_HOST`:

```shell
# Linux/macOS
export SPANNER_EMULATOR_HOST=localhost:9010

# Windows
set SPANNER_EMULATOR_HOST=localhost:9010
```

## Типы колонок
