# Oracle

## Установка

```shell
npm install oracledb
```

По умолчанию [oracledb](https://github.com/oracle/node-oracledb) подключается в «тонком режиме» (thin mode). Чтобы включить «толстый режим» (thick mode), выполните инструкции по установке из
их [руководства пользователя](https://node-oracledb.readthedocs.io/en/latest/user_guide/installation.html).

## Опции источника данных

Общие опции источника данных описаны в разделе [Опции источника данных](<../02. Data Source/02. Data Source Options.md>).

- `sid` — System Identifier (SID) идентифицирует конкретный экземпляр базы данных. Например, «sales».
- `serviceName` — Service Name — это идентификатор службы базы данных. Например, `sales.us.example.com`.

## Типы колонок

`char`, `nchar`, `nvarchar2`, `varchar2`, `long`, `raw`, `long raw`, `number`, `numeric`, `float`, `dec`, `decimal`, `integer`, `int`, `smallint`, `real`, `double precision`, `date`, `timestamp`, `timestamp with time zone`, `timestamp with local time zone`, `interval year to month`, `interval day to second`, `bfile`, `blob`, `clob`, `nclob`, `rowid`, `urowid`
