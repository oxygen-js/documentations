# Подсказки для запросов

Подсказки для запросов (query hints) — это инструкции, передаваемые вместе с SQL-запросом, которые помогают базе данных выбрать более эффективную стратегию выполнения.

Разные СУБД поддерживают разные виды подсказок: например, рекомендацию использовать индекс или выбор подходящего типа JOIN.

```typescript
await dataSource.query(`
    SELECT /*+ MAX_EXECUTION_TIME(1000) */ *
    FROM user
    WHERE email = 'example@example.com'
`)
```

В примере выше `MAX_EXECUTION_TIME(1000)` предписывает MySQL прервать запрос, если он выполняется дольше 1 секунды.
