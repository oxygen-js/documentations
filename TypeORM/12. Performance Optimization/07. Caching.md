# Кэширование

Кэширование — это приём временного хранения результатов запросов или данных для использования в последующих обращениях без повторного обращения к базе данных каждый раз.

В TypeORM есть встроенная поддержка кэширования, и вы можете настроить, как оно используется.

```typescript
const users = await userRepository
    .createQueryBuilder("user")
    .cache(true) // Enable caching
    .getMany()
```

Кроме того, можно настроить длительность кэширования или использовать внешние инструменты кэширования, например Redis, для большей эффективности.

```typescript
const dataSource = new DataSource({
    type: "mysql",
    host: "localhost",
    port: 3306,
    username: "test",
    password: "test",
    database: "test",
    cache: {
        type: "redis",
        options: {
            host: "localhost",
            port: 6379,
        },
    },
})
```
