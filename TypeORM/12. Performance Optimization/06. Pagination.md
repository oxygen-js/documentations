# Постраничная выборка

Постраничная выборка — важнейший приём повышения производительности при получении больших объёмов данных. Вместо того чтобы выбирать все данные сразу, она разбивает их на небольшие страницы, снижая нагрузку на базу данных и оптимизируя использование памяти.

В TypeORM для постраничной выборки можно использовать `limit` и `offset`.

```typescript
const users = await userRepository
    .createQueryBuilder("user")
    .limit(10) // Количество записей, выбираемых на одну страницу
    .offset(20) // Пропустить первые 20 записей
    .getMany()
```

Постраничная выборка не даёт получить большой объём данных за раз, минимизируя задержки и оптимизируя использование памяти. При её реализации подумайте об использовании курсоров — они эффективнее работают с динамически меняющимися данными.
