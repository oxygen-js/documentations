# find

#### сигнатура: `find(predicate: function)`

## Испускает первый элемент, прошедший предикат, затем завершается.

---

💡 Если вам всегда нужен первый испущенный элемент независимо от условия,
попробуйте [`first()`](<10. first.md>)!

---



### Примеры

##### Пример 1: Найти клик внутри блока, повторять при клике вне блока

( [StackBlitz](https://stackblitz.com/edit/rxjs-hd63we?file=index.ts))

```js
// RxJS v6+
import { fromEvent } from 'rxjs';
import { find, repeatWhen, mapTo, startWith, filter } from 'rxjs/operators';

// ссылка на элемент
const status = document.getElementById('status');

// потоки
const clicks$ = fromEvent(document, 'click');

clicks$
  .pipe(
    find((event: any) => event.target.id === 'box'),
    mapTo('Found!'),
    startWith('Find me!'),
    // сброс при клике вне блока
    repeatWhen(() =>
      clicks$.pipe(filter((event: any) => event.target.id !== 'box'))
    )
  )
  .subscribe(message => (status.innerHTML = message));
```
