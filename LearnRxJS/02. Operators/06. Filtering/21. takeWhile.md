# takeWhile

#### сигнатура: `takeWhile(predicate: function(value, index): boolean, inclusive?: boolean): Observable`

## Испускает значения, пока заданное выражение не станет ложным.

---

💡 Когда необязательный параметр `inclusive` установлен в `true`, он также испустит
первый элемент, не прошедший предикат.

---



### Примеры

##### Пример 1: Берём значения ниже предела

(
[StackBlitz](https://stackblitz.com/edit/typescript-af3hdf?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/zanefaqexu/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/yakd4jgc/) )

```js
// RxJS v6+
import { of } from 'rxjs';
import { takeWhile } from 'rxjs/operators';

//испускает 1,2,3,4,5
const source$ = of(1, 2, 3, 4, 5);

//пропускать значения, пока значение из источника не станет больше 4, затем завершиться
source$
  .pipe(takeWhile(val => val <= 4))
  // лог: 1,2,3,4
  .subscribe(val => console.log(val));
```

##### Пример 2: (v6.4+) takeWhile с флагом inclusive

(
[StackBlitz](https://stackblitz.com/edit/typescript-3bwfup?file=index.ts&devtoolsheight=100)
)

```js
// RxJS v6.4+
import { of } from 'rxjs';
import { takeWhile, filter } from 'rxjs/operators';

const source$ = of(1, 2, 3, 9);

source$
  // с флагом inclusive также будет испущено значение, из-за которого предикат вернул false
  .pipe(takeWhile(val => val <= 3, true))
  // лог: 1, 2, 3, 9
  .subscribe(console.log);
```

##### Пример 3: Разница между `takeWhile` и [`filter`](<08. filter.md>)

(
[StackBlitz](https://stackblitz.com/edit/typescript-roozza?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/yatoqurewi/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/r497jgw3/4/) )

```js
// RxJS v6+
import { of } from 'rxjs';
import { takeWhile, filter } from 'rxjs/operators';

// испускает 3, 3, 3, 9, 1, 4, 5, 8, 96, 3, 66, 3, 3, 3
const source$ = of(3, 3, 3, 9, 1, 4, 5, 8, 96, 3, 66, 3, 3, 3);

// пропускать значения, пока значение из источника равно 3, затем завершиться
source$
  .pipe(takeWhile(it => it === 3))
  // лог: 3, 3, 3
  .subscribe(val => console.log('takeWhile', val));

source$
  .pipe(filter(it => it === 3))
  // лог: 3, 3, 3, 3, 3, 3, 3
  .subscribe(val => console.log('filter', val));
```
