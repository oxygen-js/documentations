# single

#### сигнатура: `single(a: Function): Observable`

## Испускает единственный элемент, удовлетворяющий выражению.



### Примеры

##### Пример 1: Испускание первого числа, проходящего предикат

(
[StackBlitz](https://stackblitz.com/edit/typescript-qhynlr?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/solecibuza/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/26r5y90s/) )

```js
// RxJS v6+
import { from } from 'rxjs';
import { single } from 'rxjs/operators';

//испускаем (1,2,3,4,5)
const source = from([1, 2, 3, 4, 5]);
//испускаем один элемент, который соответствует предикату
const example = source.pipe(single(val => val === 4));
//вывод: 4
const subscribe = example.subscribe(val => console.log(val));
```
