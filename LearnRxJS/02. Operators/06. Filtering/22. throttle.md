# throttle

#### сигнатура: `throttle(durationSelector: function(value): Observable | Promise): Observable`

## Испускает значение на переднем фронте интервала, но подавляет новые значения, пока `durationSelector` не завершится.



### Примеры

##### Пример 1: Throttle на 2 секунды на основе второго observable

(
[StackBlitz](https://stackblitz.com/edit/typescript-pfcmjw?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/wohefujipo/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/h8na4m0p/) )

```js
// RxJS v6+
import { interval } from 'rxjs';
import { throttle } from 'rxjs/operators';

//испускает значение каждую секунду
const source = interval(1000);
//throttle на 2 секунды, испускает последнее значение
const example = source.pipe(throttle(val => interval(2000)));
//вывод: 0...3...6...9
const subscribe = example.subscribe(val => console.log(val));
```

##### Пример 2: Throttle с промисом

(
[StackBlitz](https://stackblitz.com/edit/typescript-g74v3q?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/seyaguwunu/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/w5Lbzz9f/) )

```js
// RxJS v6+
import { interval } from 'rxjs';
import { throttle, map } from 'rxjs/operators';

//испускает значение каждую секунду
const source = interval(1000);
//постепенно увеличиваем время разрешения промиса в зависимости от источника
const promise = val =>
  new Promise(resolve =>
    setTimeout(() => resolve(`Resolved: ${val}`), val * 100)
  );
//когда промис разрешается, испускаем элемент из источника
const example = source.pipe(
  throttle(promise),
  map(val => `Throttled off Promise: ${val}`)
);

const subscribe = example.subscribe(val => console.log(val));
```
