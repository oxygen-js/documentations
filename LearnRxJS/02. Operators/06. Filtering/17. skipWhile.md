# skipWhile

#### сигнатура: `skipWhile(predicate: Function): Observable`

## Пропускает значения из источника, пока переданное выражение не станет ложным.



### Примеры

##### Пример 1: Пропуск, пока значения ниже порога

(
[StackBlitz](https://stackblitz.com/edit/typescript-p5kapz?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/bemikuleya/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/3ymfxb09/) )

```js
// RxJS v6+
import { interval } from 'rxjs';
import { skipWhile } from 'rxjs/operators';

//испускает значение каждую секунду
const source = interval(1000);
//пропускаем значения из источника, пока значение меньше 5
const example = source.pipe(skipWhile(val => val < 5));
//вывод: 5...6...7...8........
const subscribe = example.subscribe(val => console.log(val));
```
