# takeLast

#### сигнатура: `takeLast(count: number): Observable`

## Испускает последние n испущенных значений перед завершением

---

💡 Если вам нужно только последнее испущенное значение из нескольких observable,
по завершении всех этих observable, попробуйте [forkJoin](<../01. Combination/06. forkJoin.md>)!

---



### Примеры

##### Пример 1: взять последние 2 испущенных значения перед завершением

(
[StackBlitz](https://stackblitz.com/edit/typescript-zss7oo?file=index.ts&devtoolsheight=100)
)

```js
// RxJS v6+
import { of } from 'rxjs';
import { takeLast } from 'rxjs/operators';

const source = of('Ignore', 'Ignore', 'Hello', 'World!');
// взять последние 2 испущенных значения
const example = source.pipe(takeLast(2));
// Hello, World!
const subscribe = example.subscribe(val => console.log(val));
```
