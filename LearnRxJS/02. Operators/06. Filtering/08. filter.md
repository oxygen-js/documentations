# filter

#### сигнатура: `filter(select: Function, thisArg: any): Observable`

## Испускает значения, проходящие заданное условие.

---

💡 Если вы хотите завершить observable, когда условие перестанет выполняться, обратите внимание на
[takeWhile](<21. takeWhile.md>)!

---

### Зачем нужен `filter`?

Этот оператор — то, что нужно, когда требуется отсеять нежелательные значения из потока observable. Представьте его как сеть рыбака, которая ловит только ту рыбу, что вам нужна, позволяя остальной ускользнуть.

**Ключевой момент, который стоит запомнить**, — `filter` испускает только те значения, которые удовлетворяют заданному условию. Если ни одно значение в observable не удовлетворяет условию, ничего не испускается. Это строгий вышибала на входе в клуб, пропускающий внутрь лишь тех, кто соответствует критериям.

Кроме того, для случаев, когда вы хотите не только отфильтровать значения, но и преобразовать их, [`map`](<../07. Transformation/11. map.md>) — идеальный спутник для `filter`. Используйте их в паре, чтобы одновременно формировать и очищать ваши потоки данных.



### Примеры

##### Пример 1: фильтрация чётных чисел

(
[StackBlitz](https://stackblitz.com/edit/typescript-4g4cys?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/vafogoluye/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/tkz0fuy2/) )

```js
// RxJS v6+
import { from } from 'rxjs';
import { filter } from 'rxjs/operators';

// испускаем (1,2,3,4,5)
const source = from([1, 2, 3, 4, 5]);
// отфильтровываем нечётные числа
const example = source.pipe(filter(num => num % 2 === 0));
// вывод: "Even number: 2", "Even number: 4"
const subscribe = example.subscribe(val => console.log(`Even number: ${val}`));
```

##### Пример 2: фильтрация объектов по свойству

(
[StackBlitz](https://stackblitz.com/edit/typescript-n73fsn?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/qihagaxuso/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/yjdsoug1/) )

```js
// RxJS v6+
import { from } from 'rxjs';
import { filter } from 'rxjs/operators';

// испускаем ({name: 'Joe', age: 31}, {name: 'Bob', age:25})
const source = from([
  { name: 'Joe', age: 31 },
  { name: 'Bob', age: 25 }
]);
// отфильтровываем людей младше 30
const example = source.pipe(filter(person => person.age >= 30));
// вывод: "Over 30: Joe"
const subscribe = example.subscribe(val => console.log(`Over 30: ${val.name}`));
```

##### Пример 3: фильтрация чисел больше заданного значения

(
[StackBlitz](https://stackblitz.com/edit/typescript-eyvvfu?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/rakabaheyu/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/g1tgreha/) )

```js
// RxJS v6+
import { interval } from 'rxjs';
import { filter } from 'rxjs/operators';

// испускаем каждую секунду
const source = interval(1000);
// отфильтровываем все значения, пока значение интервала не станет больше 5
const example = source.pipe(filter(num => num > 5));
/*
  "Number greater than 5: 6"
  "Number greater than 5: 7"
  "Number greater than 5: 8"
  "Number greater than 5: 9"
*/
const subscribe = example.subscribe(val =>
  console.log(`Number greater than 5: ${val}`)
);
```
