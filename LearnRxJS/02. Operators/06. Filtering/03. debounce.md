# debounce

#### сигнатура: `debounce(durationSelector: function): Observable`

## Отбрасывает испущенные значения, между которыми проходит меньше указанного времени, определяемого функцией-селектором, до следующего вывода.

---

💡 Хотя **debounce** используется не так широко, как [debounceTime](<04. debounceTime.md>), он
важен, когда интервал debounce переменный!

---



### Примеры

##### Пример 1: Debounce по таймеру

(
[StackBlitz](https://stackblitz.com/edit/typescript-dzjbra?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/sorimeyoro/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/e5698yow/) )

```js
// RxJS v6+
import { of, timer } from 'rxjs';
import { debounce } from 'rxjs/operators';

//испускаем четыре строки
const example = of('WAIT', 'ONE', 'SECOND', 'Last will display');
/*
    Испускаем значения только после того, как между последним испусканием прошла секунда,
    все остальные значения отбрасываем
*/
const debouncedExample = example.pipe(debounce(() => timer(1000)));
/*
    В этом примере все значения, кроме последнего, будут опущены
    вывод: 'Last will display'
*/
const subscribe = debouncedExample.subscribe(val => console.log(val));
```

##### Пример 2: Debounce с нарастающим интервалом

(
[StackBlitz](https://stackblitz.com/edit/typescript-qnfidr?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/sotaretese/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/6ab34nq6/) )

```js
// RxJS v6+
import { interval, timer } from 'rxjs';
import { debounce } from 'rxjs/operators';

//испускаем значение каждую 1 секунду, напр. 0...1...2
const interval$ = interval(1000);
//увеличиваем время debounce на 200 мс каждую секунду
const debouncedInterval = interval$.pipe(debounce(val => timer(val * 200)));
/*
  Через 5 секунд время debounce превысит интервал,
  все последующие значения будут отброшены
  вывод: 0...1...2...3...4......(время debounce превышает 1 с, значения не испускаются)
*/
const subscribe = debouncedInterval.subscribe(val =>
  console.log(`Example Two: ${val}`)
);
```
