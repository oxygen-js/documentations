# ignoreElements

#### сигнатура: `ignoreElements(): Observable`

## Игнорирует всё, кроме завершения и ошибки.



### Примеры

##### Пример 1: Игнорирование всех элементов источника

(
[StackBlitz](https://stackblitz.com/edit/typescript-w6rndwxh?file=index.ts))

```js
// RxJS v6+
import { interval } from 'rxjs';
import { take, ignoreElements } from 'rxjs/operators';

//испускаем значение каждые 100 мс
const source = interval(100);
//игнорируем всё, кроме завершения
const example = source.pipe(take(5), ignoreElements());
//вывод: "COMPLETE!"
const subscribe = example.subscribe(
  val => console.log(`NEXT: ${val}`),
  val => console.log(`ERROR: ${val}`),
  () => console.log('COMPLETE!')
);
```

##### Пример 2: Отображение только ошибки

(
[StackBlitz](https://stackblitz.com/edit/typescript-3yxv9z?file=index.ts&devtoolsheight=100))

```js
// RxJS v6+
import { interval, throwError, of } from 'rxjs';
import { mergeMap, ignoreElements } from 'rxjs/operators';

//испускаем значение каждые 100 мс
const source = interval(100);
//игнорируем всё, кроме ошибки
const error = source.pipe(
  mergeMap(val => {
    if (val === 4) {
      return throwError(`ERROR AT ${val}`);
    }
    return of(val);
  }),
  ignoreElements()
);
//вывод: "ERROR: ERROR AT 4"
const subscribe = error.subscribe(
  val => console.log(`NEXT: ${val}`),
  val => console.log(`ERROR: ${val}`),
  () => console.log('SECOND COMPLETE!')
);
```
