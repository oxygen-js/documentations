# skipUntil

####сигнатура: `skipUntil(the: Observable): Observable`

## Пропускает испущенные из источника значения до тех пор, пока переданный observable не испустит значение.



### Примеры

##### Пример 1: Пропуск до момента, пока observable не испустит значение

(
[StackBlitz](https://stackblitz.com/edit/typescript-gs4mps?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/tapizososu/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/xLu8nf77/) )

```js
// RxJS v6+
import { interval, timer } from 'rxjs';
import { skipUntil } from 'rxjs/operators';

//испускает каждую 1s
const source = interval(1000);
//пропускать испущенные из источника значения до тех пор, пока внутренний observable не испустит значение (6s)
const example = source.pipe(skipUntil(timer(6000)));
//вывод: 5...6...7...8........
const subscribe = example.subscribe(val => console.log(val));
```
