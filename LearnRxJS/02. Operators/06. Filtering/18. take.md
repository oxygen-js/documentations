# take

#### сигнатура: `take(count: number): Observable`

## Испускает заданное количество значений, после чего завершается.

### Зачем нужен `take`?

Когда вас интересует только первое испускание, стоит использовать `take`.
Возможно, вы хотите увидеть, на что пользователь кликнул первым при входе на страницу,
или подписаться на событие клика и взять лишь первое испускание. Ещё один сценарий
использования — когда вам нужно сделать снимок данных в конкретный момент времени,
но последующие испускания вам не требуются. Например, поток обновлений пользовательского
токена или защита маршрута (route guard) на основе потока в приложении Angular.

---

💡 Если вы хотите взять переменное количество значений на основе некоторой логики или
другого observable, можно использовать [takeUntil](<20. takeUntil.md>) или
[takeWhile](<21. takeWhile.md>)!

💡 `take` — противоположность [`skip`](<15. skip.md>): `take` возьмёт первые _n_
испусканий, тогда как `skip` пропустит первые _n_ испусканий.

### Примеры

##### Пример 1: Взять 1 значение из источника

(
[StackBlitz](https://stackblitz.com/edit/typescript-uk92ax?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/vaxitupiwi/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/f9bz0tr3/) )

```js
// RxJS v6+
import { of } from 'rxjs';
import { take } from 'rxjs/operators';

//испускаем 1,2,3,4,5
const source = of(1, 2, 3, 4, 5);
//берём первое испущенное значение, затем завершаемся
const example = source.pipe(take(1));
//вывод: 1
const subscribe = example.subscribe(val => console.log(val));
```

##### Пример 2: Взять первые 5 значений из источника

(
[StackBlitz](https://stackblitz.com/edit/typescript-3ujuth?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/kexenuzulu/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/g1fhxgua/) )

```js
// RxJS v6+
import { interval } from 'rxjs';
import { take } from 'rxjs/operators';

//испускаем значение каждую 1 с
const interval$ = interval(1000);
//берём первые 5 испущенных значений
const example = interval$.pipe(take(5));
//вывод: 0,1,2,3,4
const subscribe = example.subscribe(val => console.log(val));
```

##### Пример 3: Получение места первого клика

([StackBlitz](https://stackblitz.com/edit/typescript-8g9xt5?file=index.ts&devtoolsheight=50)
| [jsFiddle](https://jsfiddle.net/ElHuy/9c5j064x/))

```html
<div id="locationDisplay">
  Where would you click first?
</div>
```

```js
// RxJS v6+
import { fromEvent } from 'rxjs';
import { take, tap } from 'rxjs/operators';

const oneClickEvent = fromEvent(document, 'click').pipe(
  take(1),
  tap(v => {
    document.getElementById(
      'locationDisplay'
    ).innerHTML = `Your first click was on location ${v.screenX}:${v.screenY}`;
  })
);

const subscribe = oneClickEvent.subscribe();
```
