# defaultIfEmpty

#### сигнатура: `defaultIfEmpty(defaultValue: any): Observable`

## Испустить заданное значение, если до завершения ничего не было испущено.



### Примеры

##### Пример 1: Значение по умолчанию для пустого потока

(
[Stackblitz](https://stackblitz.com/edit/typescript-3btzml?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/yawumoqatu/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/8ex96cov/) )

```js
// RxJS v6+
import { defaultIfEmpty } from 'rxjs/operators';
import { of } from 'rxjs';

// испустить 'Observable.of() Empty!' если пусто, иначе любые значения от источника
const exampleOne = of().pipe(defaultIfEmpty('Observable.of() Empty!'));
// вывод: 'Observable.of() Empty!'
const subscribe = exampleOne.subscribe(val => console.log(val));
```

##### Пример 2: Значение по умолчанию для Observable.empty

(
[Stackblitz](https://stackblitz.com/edit/typescript-tyfjhu?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/kojafuvesu/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/3edw828p/) )

```js
// RxJS v6+
import { defaultIfEmpty } from 'rxjs/operators';
import { empty } from 'rxjs';

// испустить 'Observable.empty()!' если пусто, иначе любые значения от источника
const example = empty().pipe(defaultIfEmpty('Observable.empty()!'));
// вывод: 'Observable.empty()!'
const subscribe = example.subscribe(val => console.log(val));
```
