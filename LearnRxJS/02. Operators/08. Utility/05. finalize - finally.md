# finalize / finally

#### сигнатура: `finalize(callback: () => void)`

## Вызывает функцию, когда observable завершается или завершается с ошибкой

### Примеры

##### Пример 1: Выполнение функции обратного вызова при завершении observable

( [StackBlitz](https://stackblitz.com/edit/typescript-ohddud) )

```js
import { interval } from 'rxjs';
import { take, finalize } from 'rxjs/operators';

//испускаем значения по порядку каждую 1 секунду
const source = interval(1000);
//вывод: 0,1,2,3,4,5....
const example = source.pipe(
  take(5), //берём только первые 5 значений
  finalize(() => console.log('Sequence complete')) // Выполняется при завершении observable
)
const subscribe = example.subscribe(val => console.log(val));
```
