# tap / do

#### сигнатура: `tap(nextOrObserver: function, error: function, complete: function): Observable`

## Прозрачно выполняет действия или побочные эффекты, например логирование.

---

💡 Если вы используете старую версию RxJS, `tap` раньше был известен как `do`!

---

### Зачем нужен `tap`?

Представьте `tap` как камеру наблюдения в торговом центре. Она не мешает покупателям (значениям) перемещаться, а лишь наблюдает и фиксирует их действия. Этот оператор лучше всего подходит для побочных эффектов: действий, которые вы хотите выполнить в ответ на значения в observable, не влияя на сами значения.

Одна из суперспособностей `tap` — его польза при отладке. **Когда с вашим observable что-то идёт не по плану**, вместо того чтобы разбирать всю цепочку или вставлять множество логов, просто добавьте несколько операторов `tap`. Это как расставить контрольные точки в видеоигре, помогая быстро находить проблемы, не нарушая основной ход выполнения.

Однако одно предостережение: **помните, что `tap` предназначен исключительно для побочных эффектов**. Если у вас возникает соблазн изменить данные внутри `tap`, лучше воздержаться. Это не его задача, и в таких случаях лучше воспользоваться [`map`](<../07. Transformation/11. map.md>) или другими преобразующими операторами.

Наконец, стоит следить за тем, чтобы вводимые через `tap` побочные эффекты не были критичны для основной логики вашей цепочки observable, оставаясь ненавязчивыми и безвредными.



### Примеры

##### Пример 1: Логирование с помощью tap

(
[StackBlitz](https://stackblitz.com/edit/typescript-cd2gjp?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/jimazuriva/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/qtyakorq/) )

```js
// RxJS v6+
import { of } from 'rxjs';
import { tap, map } from 'rxjs/operators';

const source = of(1, 2, 3, 4, 5);
// прозрачно логируем значения из source с помощью 'tap'
const example = source.pipe(
  tap(val => console.log(`BEFORE MAP: ${val}`)),
  map(val => val + 10),
  tap(val => console.log(`AFTER MAP: ${val}`))
);

// 'tap' не преобразует значения
// вывод: 11...12...13...14...15
const subscribe = example.subscribe(val => console.log(val));
```

##### Пример 2: Использование tap с объектом

(
[StackBlitz](https://stackblitz.com/edit/typescript-3xykpb?file=index.ts&devtoolsheight=100))

```js
// RxJS v6+
import { of } from 'rxjs';
import { tap, map } from 'rxjs/operators';

const source = of(1, 2, 3, 4, 5);

// tap также принимает объект-карту для логирования next, error и complete
const example = source
  .pipe(
    map(val => val + 10),
    tap({
      next: val => {
        // при next 11 и т.д.
        console.log('on next', val);
      },
      error: error => {
        console.log('on error', error.message);
      },
      complete: () => console.log('on complete')
    })
  )
  // вывод: 11, 12, 13, 14, 15
  .subscribe(val => console.log(val));
```
