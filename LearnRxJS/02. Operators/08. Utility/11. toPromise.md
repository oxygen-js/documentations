# toPromise

#### сигнатура: `toPromise() : Promise`

## Преобразовать observable в промис.

---

⚠ `toPromise` не является пайпируемым оператором, так как не возвращает observable.

---



### Примеры

##### Пример 1: базовый Promise

( [jsBin](http://jsbin.com/favoqecixi/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/thykc9up/) )

```js
//возвращаем базовый observable
const sample = val => Rx.Observable.of(val).delay(5000);
//преобразуем базовый observable в промис
const example = sample('First Example')
  .toPromise()
  //вывод: 'First Example'
  .then(result => {
    console.log('From Promise:', result);
  });
```

##### Пример 2: использование Promise.all

( [jsBin](http://jsbin.com/hutiyicaco/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/xzu6u7hs/) )

```js
//возвращаем базовый observable
const sample = val => Rx.Observable.of(val).delay(5000);
/*
  преобразуем каждый в промис и используем Promise.all,
  чтобы дождаться разрешения всех
*/
const example = () => {
  return Promise.all([
    sample('Promise 1').toPromise(),
    sample('Promise 2').toPromise()
  ]);
};
//вывод: ["Promise 1", "Promise 2"]
example().then(val => {
  console.log('Promise.all Result:', val);
});
```
