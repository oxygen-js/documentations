# repeat

#### сигнатура: `repeat(count: number): Observable`

## Повторяет observable при завершении.

---

💡 Как [`retry`](<../04. Error Handling/02. retry.md>), но для случаев без ошибок!

---



### Примеры

##### Пример 1: Повторить 3 раза

(
[StackBlitz](https://stackblitz.com/edit/rxjs-repeat-learnrxjs?file=index.ts&devtoolsheight=100)
)

```js
// RxJS v6+
import { repeat, delay } from 'rxjs/operators';
import { of } from 'rxjs';

const delayedThing = of('delayed value').pipe(delay(2000));

delayedThing
  .pipe(repeat(3))
  // delayed value...delayed value...delayed value
  .subscribe(console.log);
```
