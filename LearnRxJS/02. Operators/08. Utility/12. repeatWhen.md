# repeatWhen

#### сигнатура: `repeatWhen(notifier: (notifications: Observable) => Observable): Observable`

## Повторяет observable при завершении на основе пользовательских критериев.

---

💡 Если вам нужно просто повторить заданное число раз, попробуйте
[retry](<../04. Error Handling/02. retry.md>)!

---



### Примеры

##### Пример 1: Повтор по интервалу

(
[StackBlitz](https://stackblitz.com/edit/rxjs-repeatwhen?file=index.ts&devtoolsheight=100)
)

```js
// RxJS v6+
import { of, interval } from 'rxjs';
import { repeatWhen, take } from 'rxjs/operators';

const repeatInterval$ = interval(1000).pipe(take(5));
const source$ = of('hey!').pipe(repeatWhen(_ => repeatInterval$));

source$.subscribe(console.log);

/*
OUTPUT:
hey!
hey!
hey!
hey!
hey!
hey!
*/
```
