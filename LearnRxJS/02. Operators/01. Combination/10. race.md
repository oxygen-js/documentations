# race

#### сигнатура: `race(): Observable`

## Используется тот observable, который испустит значение первым.

### Зачем нужен race?
Оператор `race` — то, что нужно, когда вы работаете с несколькими observable, которые соперничают друг с другом, и вас интересует только тот из них, который первым испустит значение. Это как соревновательный забег, где первый бегун, пересёкший финишную черту, объявляется победителем, а остальные уже не важны (кто не первый, тот последний?).

Понятный пример использования `race` можно увидеть в сценарии загрузки изображения. Представьте, что у вас есть два источника, из которых можно загрузить изображение, и вы хотите показать его как можно скорее. С помощью оператора `race` вы можете подписаться на оба источника, и как только первый из них успешно загрузит изображение, он испустит значение, а подписка на другой источник будет автоматически отменена.

Важно помнить, что `race` обращает внимание только на первое испущенное значение среди соперничающих observable. Как только один из observable выигрывает гонку, остальные отбрасываются, и их возможные будущие испускания никак не влияют на вывод.

Если же ваш сценарий предполагает работу с несколькими observable, которые все должны испустить значения и завершиться, либо вам нужно обрабатывать испущенные значения в определённом порядке, рассмотрите вместо этого операторы вроде [combineLatest](<02. combineLatest.md>) или [forkJoin](<06. forkJoin.md>).



### Примеры

##### Пример 1: race с 4 observable

(
[StackBlitz](https://stackblitz.com/edit/typescript-cvfmug?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/goqiwobeno/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/8jcmb1ec/) )

```js
// RxJS v6+
import { mapTo } from 'rxjs/operators';
import { interval } from 'rxjs/observable/interval';
import { race } from 'rxjs/observable/race';

//берём первый observable, который испустит значение
const example = race(
  //испускает каждые 1.5с
  interval(1500),
  //испускает каждую 1с
  interval(1000).pipe(mapTo('1s won!')),
  //испускает каждые 2с
  interval(2000),
  //испускает каждые 2.5с
  interval(2500)
);
//вывод: "1s won!"..."1s won!"...и т.д.
const subscribe = example.subscribe(val => console.log(val));
```

##### Пример 2: race с ошибкой

(
[StackBlitz](https://stackblitz.com/edit/typescript-in6fw6?file=index.ts&devtoolsheight=100)
| [jsFiddle](https://jsfiddle.net/gbeL4t55/2/) )

```js
// RxJS v6+
import { delay, map } from 'rxjs/operators';
import { of, race } from 'rxjs';

//Выбрасывает ошибку и игнорирует остальные observable.
const first = of('first').pipe(
  delay(100),
  map(_ => {
    throw 'error';
  })
);
const second = of('second').pipe(delay(200));
const third = of('third').pipe(delay(300));
// ничего не выводится
race(first, second, third).subscribe(val => console.log(val));
```
