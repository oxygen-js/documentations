# startWith

#### сигнатура: `startWith(an: Values): Observable`

## Сначала испускает заданное значение.

---

💡 [BehaviorSubject](https://github.com/Reactive-Extensions/RxJS/blob/master/doc/api/subjects/behaviorsubject.md)
тоже может начинаться с начального значения!

---

### Зачем нужен startWith?
Оператор `startWith` — отличный инструмент, когда нужно задать начальное значение для последовательности observable, гарантируя, что потребитель всегда получит значение при подписке. Это удобный способ задать состояние или значение по умолчанию для ваших observable, упрощая подписчикам работу с данными и снижая вероятность столкнуться с неожиданными ситуациями.

Пример из реальной жизни — функция поиска, где результаты должны показывать список популярных элементов в качестве состояния по умолчанию, пока пользователь не начал вводить свой запрос. С помощью startWith вы можете беспрепятственно предоставить эти данные по умолчанию своим подписчикам.

Имейте в виду, что startWith испускает начальное значение немедленно при подписке. Такое поведение полезно, когда нужно, чтобы подписчики получили значение сразу же, ещё до того, как исходный observable начнёт испускать значения.

##### Пример 1: startWith на числовой последовательности

(
[StackBlitz](https://stackblitz.com/edit/typescript-2qrwjt?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/lezuravizu/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/e8dn3ggp/) )

```js
// RxJS v6+
import { startWith } from 'rxjs/operators';
import { of } from 'rxjs';

//испускает (1,2,3)
const source = of(1, 2, 3);
//начать с 0
const example = source.pipe(startWith(0));
//вывод: 0,1,2,3
const subscribe = example.subscribe(val => console.log(val));
```

##### Пример 2: startWith для начального значения scan

(
[StackBlitz](https://stackblitz.com/edit/typescript-8gkbsc?file=index.ts&devtoolsheight=100)
| | [jsBin](http://jsbin.com/gemevuzoha/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/54r3g83e/) )

```js
// RxJS v6+
import { startWith, scan } from 'rxjs/operators';
import { of } from 'rxjs';

//испускает ('World!', 'Goodbye', 'World!')
const source = of('World!', 'Goodbye', 'World!');
//начать с 'Hello', присоединять текущую строку к предыдущей
const example = source.pipe(
  startWith('Hello'),
  scan((acc, curr) => `${acc} ${curr}`)
);
/*
  вывод:
  "Hello"
  "Hello World!"
  "Hello World! Goodbye"
  "Hello World! Goodbye World!"
*/
const subscribe = example.subscribe(val => console.log(val));
```

##### Пример 3: startWith с несколькими значениями

(
[StackBlitz](https://stackblitz.com/edit/typescript-ek45ff?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/cumupemuxa/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/ckcyj3ms/) )

```js
// RxJS v6+
import { startWith } from 'rxjs/operators';
import { interval } from 'rxjs';

//испускает значения по порядку каждую 1с
const source = interval(1000);
//начать с -3, -2, -1
const example = source.pipe(startWith(-3, -2, -1));
//вывод: -3, -2, -1, 0, 1, 2....
const subscribe = example.subscribe(val => console.log(val));
```
