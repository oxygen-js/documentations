# merge

#### сигнатура: `merge(input: Observable): Observable`

## Превращает несколько observable в единый observable.

---

💡 Этот оператор можно использовать как в качестве статического, так и в качестве метода экземпляра!

💡 Если приоритетом является порядок, а не пропускная способность, попробуйте вместо этого [concat](<03. concat.md>)!

---

### Зачем нужен merge?
Оператор merge — это ваше решение, когда у вас есть несколько observable, которые производят значения независимо друг от друга, и вы хотите объединить их вывод в единый поток. Считайте, что это слияние на автомагистрали, где несколько дорог сходятся вместе, образуя единую общую дорогу — трафик (данные) с каждой дороги (observable) плавно течёт вместе.

Пример из реальной жизни можно увидеть в чат-приложении, где у вас есть отдельные observable для получения сообщений от нескольких пользователей. Используя `merge`, вы можете свести все эти потоки сообщений в единый общий поток для отображения сообщений в окне чата.

Имейте в виду, что `merge` испускает значения, как только любой из observable испустит значение. Это отличается от combineLatest или withLatestFrom, которые ждут, пока каждый observable испустит хотя бы одно значение, прежде чем испустить объединённое значение.

Наконец, если вы имеете дело с observable, которые испускают значения через определённые интервалы, и вам нужно объединить их на основе времени, рассмотрите использование оператора [zip](<13. zip.md>).



### Примеры

##### Пример 1: слияние нескольких observable, статический метод

(
[StackBlitz](https://stackblitz.com/edit/typescript-ohq6rx?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/conufujapi/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/qvq9dscu/) )

```js
// RxJS v6+
import { mapTo } from 'rxjs/operators';
import { interval, merge } from 'rxjs';

//испускает каждые 2.5 секунды
const first = interval(2500);
//испускает каждые 2 секунды
const second = interval(2000);
//испускает каждые 1.5 секунды
const third = interval(1500);
//испускает каждую 1 секунду
const fourth = interval(1000);

//испускает вывод из одного observable
const example = merge(
  first.pipe(mapTo('FIRST!')),
  second.pipe(mapTo('SECOND!')),
  third.pipe(mapTo('THIRD')),
  fourth.pipe(mapTo('FOURTH'))
);
//вывод: "FOURTH", "THIRD", "SECOND!", "FOURTH", "FIRST!", "THIRD", "FOURTH"
const subscribe = example.subscribe(val => console.log(val));
```

##### Пример 2: слияние 2 observable, метод экземпляра

(
[StackBlitz](https://stackblitz.com/edit/typescript-bcsl1r?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/wuwujokaqu/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/me5ofcr0/) )

```js
// RxJS v6+
import { merge } from 'rxjs/operators';
import { interval } from 'rxjs';

//испускает каждые 2.5 секунды
const first = interval(2500);
//испускает каждую 1 секунду
const second = interval(1000);
//используется как метод экземпляра
const example = first.pipe(merge(second));
//вывод: 0,1,0,2....
const subscribe = example.subscribe(val => console.log(val));
```
