# combineLatest

#### сигнатура: `combineLatest(observables: ...Observable, project: function): Observable`

## Когда любой из observable испускает значение, испустить последнее испущенное значение от каждого из них.

---

💡 [combineLatestAll](<01. combineLatestAll.md>) можно использовать, чтобы применить combineLatest
к испущенным observable, когда источник завершается!

---

### Зачем нужен `combineLatest`?

Этот оператор лучше всего подходит, когда у вас есть несколько долгоживущих observable, которые
зависят друг от друга при каком-либо вычислении или принятии решения. Простые примеры этого
можно увидеть в [примере три](#пример-3-объединение-событий-от-2-кнопок),
где события от нескольких кнопок объединяются, чтобы получить счётчик каждой из них
и общий итог, или в
[вычислении индекса массы тела (BMI)](http://reactivex.io/rxjs/class/es6/Observable.js~Observable.html#instance-method-combineLatest)
из документации RxJS.

Учтите, что **`combineLatest` не испустит начальное значение до тех пор, пока каждый
observable не испустит хотя бы одно значение**. Это то же самое поведение, что и у
[`withLatestFrom`](<12. withLatestFrom.md>), и оно может стать _подвохом_, поскольку не будет
никакого вывода и никакой ошибки, но при этом один (или несколько) из ваших внутренних observable,
скорее всего, работает не так, как задумано, либо подписка запаздывает.

Наконец, если вы работаете с observable, которые испускают только одно значение, или вам
нужно лишь последнее значение каждого из них перед завершением, [`forkJoin`](<06. forkJoin.md>),
вероятно, будет лучшим вариантом.



### Примеры

##### Пример 1: Объединение observable, испускающих с 3 разными интервалами

(
[StackBlitz](https://stackblitz.com/edit/typescript-vadvm2?file=index.ts&devtoolsheight=100)
)

```js
// RxJS v6+
import { timer, combineLatest } from 'rxjs';

// timerOne испускает первое значение через 1с, затем каждые 4с
const timerOne$ = timer(1000, 4000);
// timerTwo испускает первое значение через 2с, затем каждые 4с
const timerTwo$ = timer(2000, 4000);
// timerThree испускает первое значение через 3с, затем каждые 4с
const timerThree$ = timer(3000, 4000);

// когда один из таймеров испускает, испустить последние значения от каждого таймера в виде массива
combineLatest(timerOne$, timerTwo$, timerThree$).subscribe(
  ([timerValOne, timerValTwo, timerValThree]) => {
    /*
  	Пример:
    первый тик timerThree: 'Timer One Latest: 0, Timer Two Latest: 0, Timer Three Latest: 0
    второй тик timerOne: 'Timer One Latest: 1, Timer Two Latest: 0, Timer Three Latest: 0
    второй тик timerTwo: 'Timer One Latest: 1, Timer Two Latest: 1, Timer Three Latest: 0
  */
    console.log(
      `Timer One Latest: ${timerValOne},
     Timer Two Latest: ${timerValTwo},
     Timer Three Latest: ${timerValThree}`
    );
  }
);
```

##### Пример 2: combineLatest с функцией проекции

(
[StackBlitz](https://stackblitz.com/edit/typescript-prtbvd?file=index.ts&devtoolsheight=100)
)

```js
// RxJS v6+
import { timer, combineLatest } from 'rxjs';

const timerOne$ = timer(1000, 4000);
const timerTwo$ = timer(2000, 4000);
const timerThree$ = timer(3000, 4000);

combineLatest(
  timerOne$,
  timerTwo$,
  timerThree$,
  // combineLatest также принимает необязательную функцию проекции
  (one, two, three) => {
    return `Timer One (Proj) Latest: ${one}, 
              Timer Two (Proj) Latest: ${two}, 
              Timer Three (Proj) Latest: ${three}`;
  }
).subscribe(console.log);
```

##### Пример 3: Объединение событий от 2 кнопок

(
[StackBlitz](https://stackblitz.com/edit/typescript-ihcxud?file=index.ts&devtoolsheight=50)
)

```js
// RxJS v6+
import { fromEvent, combineLatest } from 'rxjs';
import { mapTo, startWith, scan, tap, map } from 'rxjs/operators';

// ссылки на элементы
const redTotal = document.getElementById('red-total');
const blackTotal = document.getElementById('black-total');
const total = document.getElementById('total');

const addOneClick$ = id =>
  fromEvent(document.getElementById(id), 'click').pipe(
    // отобразить каждый клик в 1
    mapTo(1),
    // вести нарастающий итог
    scan((acc, curr) => acc + curr, 0),
    startWith(0)
  );

combineLatest(addOneClick$('red'), addOneClick$('black')).subscribe(
  ([red, black]: any) => {
    redTotal.innerHTML = red;
    blackTotal.innerHTML = black;
    total.innerHTML = red + black;
  }
);
```

###### HTML

```html
<div>
  <button id="red">Red</button>
  <button id="black">Black</button>
</div>
<div>Red: <span id="red-total"></span></div>
<div>Black: <span id="black-total"></span></div>
<div>Total: <span id="total"></span></div>
```
