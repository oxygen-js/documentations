# multicast

#### сигнатура: `multicast(selector: Function): Observable`

## Разделяет источник с помощью предоставленного Subject.



### Примеры

##### Пример 1: multicast со стандартным Subject

(
[StackBlitz](https://stackblitz.com/edit/typescript-vge8sk?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/zexuyosuvi/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/x2z7p1gm/) )

```js
// RxJS v6+
import { Subject, interval } from 'rxjs';
import { take, tap, multicast, mapTo } from 'rxjs/operators';

// испускаем каждые 2 секунды, берём 5
const source = interval(2000).pipe(take(5));

const example = source.pipe(
  // поскольку ниже мы делаем multicast, побочные эффекты выполнятся один раз
  tap(() => console.log('Side Effect #1')),
  mapTo('Result!')
);

// подписываем subject на source при вызове connect()
const multi = example.pipe(multicast(() => new Subject()));
/*
  подписчики будут разделять source
  вывод:
  "Side Effect #1"
  "Result!"
  "Result!"
  ...
*/
const subscriberOne = multi.subscribe(val => console.log(val));
const subscriberTwo = multi.subscribe(val => console.log(val));
// подписываем subject на source
multi.connect();
```

##### Пример 2: multicast с ReplaySubject

(
[StackBlitz](https://stackblitz.com/edit/typescript-n5ghjj?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/ruhexuhike/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/oj68u58j/) )

```js
// RxJS v6+
import { interval, ReplaySubject } from 'rxjs';
import { take, multicast, tap, mapTo } from 'rxjs/operators';

// испускаем каждые 2 секунды, берём 5
const source = interval(2000).pipe(take(5));

// пример с ReplaySubject
const example = source.pipe(
  // поскольку ниже мы делаем multicast, побочные эффекты выполнятся один раз
  tap(_ => console.log('Side Effect #2')),
  mapTo('Result Two!')
);
// можно использовать любой тип subject
const multi = example.pipe(multicast(() => new ReplaySubject(5)));
// подписываем subject на source
multi.connect();

setTimeout(() => {
  /*
   подписчик получит все предыдущие значения при подписке
   благодаря ReplaySubject
   */
  const subscriber = multi.subscribe(val => console.group(val));
}, 5000);
```
