# share

#### сигнатура: `share(): Observable`

## Разделить источник между несколькими подписчиками.

---

💡 share — это как [multicast](<02. multicast.md>) с Subject и refCount!

---



### Примеры

##### Пример 1: несколько подписчиков разделяют источник

(
[StackBlitz](https://stackblitz.com/edit/typescript-dlaa1p?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/jobiyomari/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/Lmesxxaq/) )

```js
// RxJS v6+
import { timer } from 'rxjs';
import { tap, mapTo, share } from 'rxjs/operators';

//испускаем значение через 1с
const source = timer(1000);
//логируем побочный эффект, испускаем результат
const example = source.pipe(
  tap(() => console.log('***SIDE EFFECT***')),
  mapTo('***RESULT***')
);

/*
  ***НЕ РАЗДЕЛЯЕТСЯ, ПОБОЧНЫЙ ЭФФЕКТ БУДЕТ ВЫПОЛНЕН ДВАЖДЫ***
  вывод:
  "***SIDE EFFECT***"
  "***RESULT***"
  "***SIDE EFFECT***"
  "***RESULT***"
*/
const subscribe = example.subscribe(val => console.log(val));
const subscribeTwo = example.subscribe(val => console.log(val));

//разделяем observable между подписчиками
const sharedExample = example.pipe(share());
/*
  ***РАЗДЕЛЯЕТСЯ, ПОБОЧНЫЙ ЭФФЕКТ ВЫПОЛНЕН ОДИН РАЗ***
  вывод:
  "***SIDE EFFECT***"
  "***RESULT***"
  "***RESULT***"
*/
const subscribeThree = sharedExample.subscribe(val => console.log(val));
const subscribeFour = sharedExample.subscribe(val => console.log(val));
```
