# fromEvent

#### сигнатура: `fromEvent(target: EventTargetLike, eventName: string, selector: function): Observable`

## Превращает событие в последовательность observable.



### Примеры

##### Пример 1: Observable из кликов мыши

(
[StackBlitz](https://stackblitz.com/edit/typescript-mfyefr?file=index.ts&devtoolsheight=50)
| [jsBin](http://jsbin.com/xikapewoqa/1/edit?js,console,output) |
[jsFiddle](https://jsfiddle.net/btroncone/vbLz1pdx/) )

```js
// RxJS v6+
import { fromEvent } from 'rxjs';
import { map } from 'rxjs/operators';

//создаём observable, испускающий события клика
const source = fromEvent(document, 'click');
//преобразуем в строку с временной меткой события
const example = source.pipe(map(event => `Event time: ${event.timeStamp}`));
//вывод (пример): 'Event time: 7276.390000000001'
const subscribe = example.subscribe(val => console.log(val));
```
