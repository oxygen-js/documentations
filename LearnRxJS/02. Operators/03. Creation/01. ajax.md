# ajax

#### сигнатура: `ajax(urlOrRequest: string | AjaxRequest)`

## Создаёт observable для Ajax-запроса, принимая либо объект запроса с url, headers и т.п., либо строку с URL.



### Примеры

##### Пример 1: Observable, испускающий объект ответа, который возвращается из запроса.

( [StackBlitz](https://stackblitz.com/edit/rxjs-raqi89) )

```js
// RxJS v6+
import { ajax } from 'rxjs/ajax';

const githubUsers = `https://api.github.com/users?per_page=2`;

const users = ajax(githubUsers);

const subscribe = users.subscribe(
  res => console.log(res),
  err => console.error(err)
);
```

##### Пример 2: Observable, испускающий только ключ json объекта ответа, который возвращается из запроса.

( [StackBlitz](https://stackblitz.com/edit/rxjs-8jkrhl) )

```js
// RxJS v6+
import { ajax } from 'rxjs/ajax';

const githubUsers = `https://api.github.com/users?per_page=2`;

const users = ajax.getJSON(githubUsers);

  res => console.log(res),
  err => console.error(err)
);
```

##### Пример 3: Observable, испускающий объект ошибки, который возвращается из запроса.

( [StackBlitz](https://stackblitz.com/edit/rxjs-vnxkth) )

```js
// RxJS v6+
import { ajax } from 'rxjs/ajax';

const githubUsers = `https://api.github.com/error`;

const users = ajax.getJSON(githubUsers);

const subscribe = users.subscribe(
  res => console.log(res),
  err => console.error(err)
);

/* вывод
Error: ajax error 404
*/
```

##### Пример 4: Оператор ajax с объектом на входе.

( [StackBlitz](https://stackblitz.com/edit/rxjs-vqnnot) )

```js
// RxJS v6+
import { ajax } from 'rxjs/ajax';

const githubUsers = `https://api.github.com/error`;

const users = ajax({
  url: githubUsers,
  method: 'GET',
  headers: {
    /*какие-то заголовки*/
  },
  body: {
    /*на случай, если вам нужно тело запроса*/
  }
});

const subscribe = users.subscribe(
  res => console.log(res),
  err => console.error(err)
);
```
