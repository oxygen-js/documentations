### Примеры

##### Пример 1: defer для получения текущей даты/времени в момент подписки

(
[StackBlitz](https://stackblitz.com/edit/rxjs-defer-example?file=index.ts&devtoolsheight=100)
)

```js
// RxJS v6+
import { defer, of, timer, merge } from 'rxjs';
import { switchMap } from 'rxjs/operators';

const s1 = of(new Date()); //захватит текущие дату и время
const s2 = defer(() => of(new Date())); //захватит дату и время в момент подписки

console.log(new Date());

timer(2000)
  .pipe(switchMap(_ => merge(s1, s2)))
  .subscribe(console.log);

/*
OUTPUT => 
2019-02-10T12:38:30.000Z (текущие дата/время из первого console.log)
2019-02-10T12:38:30.000Z (дата/время в console.log для s1, захвачены в момент создания observable)
2019-02-10T12:38:32.000Z (дата/время в console.log для s2, захвачены в момент подписки)
*/

/*//ПРИМЕЧАНИЕ: 'традиционный' js-эквивалент приведённого выше кода с timer:
setTimeout(() => {
  s1.subscribe(console.log);
  s2.subscribe(console.log);
}, 2000);
*/
```
