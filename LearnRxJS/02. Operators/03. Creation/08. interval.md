# interval

#### сигнатура: `interval(period: number, scheduler: Scheduler): Observable`

## Испускает числа по порядку с заданным интервалом времени.



### Примеры

##### Пример 1: Испускание последовательности значений с интервалом в 1 секунду

(
[StackBlitz](https://stackblitz.com/edit/typescript-ohddud?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/vigohomabo/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/x3mrwzr0/) )

```js
// RxJS v6+
import { interval } from 'rxjs';

//испускает значения по порядку каждую 1 секунду
const source = interval(1000);
//вывод: 0,1,2,3,4,5....
const subscribe = source.subscribe(val => console.log(val));
```
