``# from

#### сигнатура: `from(ish: ObservableInput, mapFn: function, thisArg: any, scheduler: Scheduler): Observable`

## Превращает массив, промис или итерируемый объект в observable.

---

💡 Этот оператор можно использовать для преобразования промиса в observable!

💡 Для массивов и итерируемых объектов все содержащиеся в них значения будут испущены как последовательность!

💡 Этот оператор также можно использовать, чтобы испустить строку как последовательность символов!

---



### Примеры

##### Пример 1: Observable из массива

(
[StackBlitz](https://stackblitz.com/edit/typescript-sckwsw?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/foceyuketi/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/o7kb5e6j/) )

```js
// RxJS v6+
import { from } from 'rxjs';

//испускаем массив как последовательность значений
const arraySource = from([1, 2, 3, 4, 5]);
//вывод: 1,2,3,4,5
const subscribe = arraySource.subscribe(val => console.log(val));
```

##### Пример 2: Observable из промиса

(
[StackBlitz](https://stackblitz.com/edit/typescript-clpg1f?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/tamofinujo/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/2czc5sae/) )

```js
// RxJS v6+
import { from } from 'rxjs';

//испускаем результат промиса
const promiseSource = from(new Promise(resolve => resolve('Hello World!')));
//вывод: 'Hello World'
const subscribe = promiseSource.subscribe(val => console.log(val));
```

##### Пример 3: Observable из коллекции

(
[StackBlitz](https://stackblitz.com/edit/typescript-drfckx?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/tezohobudu/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/ae6hu9a8/) )

```js
// RxJS v6+
import { from } from 'rxjs';

//работает с js-коллекциями
const map = new Map();
map.set(1, 'Hi');
map.set(2, 'Bye');

const mapSource = from(map);
//вывод: [1, 'Hi'], [2, 'Bye']
const subscribe = mapSource.subscribe(val => console.log(val));
```

##### Пример 4: Observable из строки

(
[StackBlitz](https://stackblitz.com/edit/typescript-19nejh?file=index.ts&devtoolsheight=100)

```js
// RxJS v6+
import { from } from 'rxjs';

//испускаем строку как последовательность

//вывод: 'H','e','l','l','o',' ','W','o','r','l','d'
const subscribe = source.subscribe(val => console.log(val));
```
