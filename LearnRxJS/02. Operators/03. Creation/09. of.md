# of / just

#### сигнатура: `of(...values, scheduler: Scheduler): Observable`

## Испускает переменное количество значений в последовательности, а затем испускает уведомление о завершении.

### Примеры

##### Пример 1: Испускание последовательности чисел

(
[StackBlitz](https://stackblitz.com/edit/typescript-kbpvmm?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/kodixitoji/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/f7b35ayz/) )

```js
// RxJS v6+
import { of } from 'rxjs';
//испускает любое количество переданных значений в последовательности
const source = of(1, 2, 3, 4, 5);
//вывод: 1,2,3,4,5
const subscribe = source.subscribe(val => console.log(val));
```

##### Пример 2: Испускание объекта, массива и функции

(
[StackBlitz](https://stackblitz.com/edit/typescript-m1jbw9?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/xevobujama/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/d9rng4dj/) )

```js
// RxJS v6+
import { of } from 'rxjs';
//испускает значения любого типа
const source = of({ name: 'Brian' }, [1, 2, 3], function hello() {
  return 'Hello';
});
//вывод: {name: 'Brian'}, [1,2,3], function hello() { return 'Hello' }
const subscribe = source.subscribe(val => console.log(val));
```
