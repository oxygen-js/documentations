# concatMapTo

#### сигнатура: `concatMapTo(observable: Observable, resultSelector: function): Observable`

## Подписывается на переданный observable, когда предыдущий завершается, и испускает значения.



### Примеры

##### Пример 1: Отображение в базовый observable (имитация запроса)

(
[StackBlitz](https://stackblitz.com/edit/typescript-fkkh6c?file=index.ts&devtoolsheight=50)
)

```js
// RxJS v6+
import { of, interval } from 'rxjs';
import { concatMapTo, delay, take } from 'rxjs/operators';

//испускает значение каждые 2 секунды
const sampleInterval = interval(500).pipe(take(5));
const fakeRequest = of('Network request complete').pipe(delay(3000));
//ждём завершения первого, прежде чем подписаться на следующий
const example = sampleInterval.pipe(concatMapTo(fakeRequest));
//результат
//вывод: Network request complete...3s...Network request complete'
const subscribe = example.subscribe(val => console.log(val));
```

##### Пример 2: Использование проекции с `concatMap`

(
[StackBlitz](https://stackblitz.com/edit/typescript-8kcfm1?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/fogefebisu/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/s19wtscb/) )

```js
// RxJS v6+
import { interval } from 'rxjs';
import { concatMapTo, take } from 'rxjs/operators';
//испускает значение каждые 2 секунды
const interval$ = interval(2000);
//испускает значение каждую секунду в течение 5 секунд
const source = interval(1000).pipe(take(5));
/*
  ***Будьте осторожны***: в подобных ситуациях, когда источник испускает значения быстрее,
  чем завершается внутренний observable, могут возникнуть проблемы с памятью.
  (interval испускает каждую секунду, а basicTimer завершается каждые 5)
*/
// basicTimer завершится через 5 секунд, испустив 0,1,2,3,4
const example = interval$.pipe(
  concatMapTo(
    source,
    (firstInterval, secondInterval) => `${firstInterval} ${secondInterval}`
  )
);
/*
  вывод: 0 0
          0 1
          0 2
          0 3
          0 4
          1 0
          1 1
          и так далее...

*/
const subscribe = example.subscribe(val => console.log(val));
```
