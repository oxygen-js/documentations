# windowWhen

#### сигнатура: `windowWhen(closingSelector: function(): Observable): Observable`

## Закрывает окно в заданный момент времени, испуская observable из собранных значений источника.



### Примеры

##### Пример 1: Открытие и закрытие окна по интервалу

(
[StackBlitz](https://stackblitz.com/edit/typescript-52tu8k?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/tuhaposemo/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/gnx9fb3h/) )

```js
// RxJS v6+
import { interval, timer } from 'rxjs';
import { windowWhen, tap, mergeAll } from 'rxjs/operators';

//испускаем сразу же, затем каждую 1с
const source = timer(0, 1000);
const example = source.pipe(
  //закрываем окно каждые 5с и испускаем observable из собранных значений источника
  windowWhen(() => interval(5000)),
  tap(_ => console.log('NEW WINDOW!'))
);

const subscribeTwo = example
  .pipe(
    //окно испускает вложенный observable
    mergeAll()
    /*
      вывод:
      "NEW WINDOW!"
      0
      1
      2
      3
      4
      "NEW WINDOW!"
      5
      6
      7
      8
      9
    */
  )
  .subscribe(val => console.log(val));
```
