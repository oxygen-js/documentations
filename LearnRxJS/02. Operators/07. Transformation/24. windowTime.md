# windowTime

#### сигнатура: `windowTime(windowTimeSpan: number, windowCreationInterval: number, scheduler: Scheduler): Observable`

## Observable из значений, собранных от источника за каждый заданный промежуток времени.



### Примеры

##### Пример 1: Открывать новое окно каждый указанный промежуток времени

(
[StackBlitz](https://stackblitz.com/edit/typescript-hkrjsc?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/mifayacoqo/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/g04b3qeb/) )

```js
// RxJS v6+
import { timer } from 'rxjs';
import { windowTime, tap, mergeAll } from 'rxjs/operators';

// испустить немедленно, затем каждую 1с
const source = timer(0, 1000);
const example = source.pipe(
  // начинать новое окно каждые 3с
  windowTime(3000),
  tap(_ => console.log('NEW WINDOW!'))
);

const subscribeTwo = example
  .pipe(
    // окно испускает вложенный observable
    mergeAll()
    /*
            вывод:
            "NEW WINDOW!"
            0
            1
            2
            "NEW WINDOW!"
            3
            4
            5
          */
  )
  .subscribe(val => console.log(val));
```
