# bufferTime

#### сигнатура: `bufferTime(bufferTimeSpan: number, bufferCreationInterval: number, scheduler: Scheduler): Observable`

## Собирает испущенные значения, пока не пройдёт заданное время, затем испускает их в виде массива.

### Зачем нужен `bufferTime`?
Ключевое отличие `bufferTime` от других операторов буферизации заключается в его подходе к буферизации на основе времени. `bufferTime` накапливает значения из исходного observable в массив в течение указанного промежутка времени, прежде чем испустить буферизованный массив.

Этот оператор особенно хорошо подходит для сценариев, где нужно пакетировать или притормаживать эмиссии из высокочастотных observable, например при отслеживании действий пользователя, движений мыши или работе с быстро обновляющимися потоками данных. `bufferTime` предоставляет эффективный способ обрабатывать эти эмиссии в более управляемой, привязанной ко времени манере.

Помните, что `bufferTime` позволяет эффективно управлять эмиссией данных, собирая и испуская их привязанными ко времени пакетами, как показано в первом примере. Однако учитывайте его особенности и выбирайте правильный оператор в соответствии с вашим конкретным сценарием использования.

### Примеры

##### Пример 1: Буферизация в течение 2 секунд

( [StackBlitz](https://stackblitz.com/edit/typescript-haqxd1?file=index.ts&devtoolsheight=50))

```js
// RxJS v6+
import { interval } from 'rxjs';
import { bufferTime } from 'rxjs/operators';

//Создаём observable, который испускает значение каждые 500 мс
const source = interval(500);
//По прошествии 2 секунд испускаем буферизованные значения в виде массива
const example = source.pipe(bufferTime(2000));
//Выводим значения в консоль
//напр. вывод [0,1,2]...[3,4,5,6]
const subscribe = example.subscribe(val =>
  console.log('Buffered with Time:', val)
);
```

##### Пример 2: Несколько активных буферов

( [StackBlitz](https://stackblitz.com/edit/typescript-9blquz?file=index.ts&devtoolsheight=100))

```js
// RxJS v6+
import { interval } from 'rxjs';
import { bufferTime } from 'rxjs/operators';

//Создаём observable, который испускает значение каждые 500 мс
const source = interval(500);
/*
bufferTime также принимает второй аргумент — когда начинать следующий буфер (время в мс)
например, если у нас bufferTime равен 2 секундам, а второй аргумент (bufferCreationInterval) — 1 секунде:
напр. вывод: [0,1,2]...[1,2,3,4,5]...[3,4,5,6,7]
*/
const example = source.pipe(bufferTime(2000, 1000));
//Выводим значения в консоль
const subscribe = example.subscribe(val =>
  console.log('Start Buffer Every 1s:', val)
);
```
