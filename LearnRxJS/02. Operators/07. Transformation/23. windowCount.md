# windowCount

#### сигнатура: `windowCount(windowSize: number, startWindowEvery: number): Observable`

## Observable значений из источника, испускаемый каждый раз, когда достигается заданное количество.



### Примеры

##### Пример 1: Начинать новое окно каждые x испущенных элементов

(
[StackBlitz](https://stackblitz.com/edit/typescript-kcxi8y?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/nezuvacexe/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/xjgbnqp5/) )

```js
// RxJS v6+
import { interval } from 'rxjs';
import { windowCount, mergeAll, tap } from 'rxjs/operators';

//испускает каждую 1с
const source = interval(1000);
const example = source.pipe(
  //начинать новое окно каждые 4 испущенных значения
  windowCount(4),
  tap(_ => console.log('NEW WINDOW!'))
);

const subscribeTwo = example
  .pipe(
    //окно испускает вложенный observable
    mergeAll()
    /*
            вывод:
            "NEW WINDOW!"
            0
            1
            2
            3
            "NEW WINDOW!"
            4
            5
            6
            7
          */
  )
  .subscribe(val => console.log(val));
```
