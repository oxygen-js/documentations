# bufferWhen

#### сигнатура: `bufferWhen(closingSelector: function): Observable`

## Собирает все значения, пока не испустит закрывающий селектор, затем испускает накопленные значения.



### Примеры

##### Пример 1: Испускание буфера по интервалу

(
[StackBlitz](https://stackblitz.com/edit/typescript-f4a2fu?file=index.ts&devtoolsheight=10)
| [jsBin](http://jsbin.com/vugerupube/1/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/nr9agfuL/) )

```js
// RxJS v6+
import { interval } from 'rxjs';
import { bufferWhen } from 'rxjs/operators';

//испускаем значение каждую 1 секунду
const oneSecondInterval = interval(1000);
//возвращаем observable, который испускает значение каждые 5 секунд
const fiveSecondInterval = () => interval(5000);
//каждые пять секунд испускаем накопленные значения
const bufferWhenExample = oneSecondInterval.pipe(
  bufferWhen(fiveSecondInterval)
);
//выводим значения
//пример вывода: [0,1,2,3]...[4,5,6,7,8]
const subscribe = bufferWhenExample.subscribe(val =>
  console.log('Emitted Buffer: ', val)
);
```
