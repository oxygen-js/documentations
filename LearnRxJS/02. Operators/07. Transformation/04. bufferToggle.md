# bufferToggle

#### сигнатура: `bufferToggle(openings: Observable, closingSelector: Function): Observable`

## Включается, чтобы перехватывать значения из источника, выключается, чтобы испустить буферизованные значения в виде массива.



### Примеры

##### Пример 1: Включение и выключение буфера по интервалу

(
[StackBlitz](https://stackblitz.com/edit/typescript-xu3sq8?file=index.ts&devtoolsheight=100)
| [jsBin](http://jsbin.com/relavezugo/edit?js,console) |
[jsFiddle](https://jsfiddle.net/btroncone/6ad3w3wf/) )

```js
// RxJS v6+
import { interval } from 'rxjs';
import { bufferToggle } from 'rxjs/operators';

//испускает значение каждую секунду
const sourceInterval = interval(1000);
//запускает первый буфер через 5 секунд и затем каждые 5 секунд
const startInterval = interval(5000);
//испускает значение через 3 секунды, закрывая соответствующий буфер
const closingInterval = val => {
  console.log(`Value ${val} emitted, starting buffer! Closing in 3s!`);
  return interval(3000);
};
//каждые 5 секунд запускается новый буфер, собирающий испущенные значения в течение 3 секунд, после чего испускает буферизованные значения
const bufferToggleInterval = sourceInterval.pipe(
  bufferToggle(startInterval, closingInterval)
);
//вывод в консоль
//напр. испущенные буферы [4,5,6]...[9,10,11]
const subscribe = bufferToggleInterval.subscribe(val =>
  console.log('Emitted Buffer:', val)
);
```

##### Пример 2: Включение и выключение буфера при нажатии/отпускании кнопки мыши

(
[StackBlitz](https://stackblitz.com/edit/rxjs-buffertoggle-mousemove?file=index.ts&devtoolsheight=50)
)

```js
import { fromEvent } from 'rxjs';
import { bufferToggle } from 'rxjs/operators';

fromEvent(document, 'mousemove')
  .pipe(
    bufferToggle(fromEvent(document, 'mousedown'), _ =>
      fromEvent(document, 'mouseup')
    )
  )
  .subscribe(console.log);
```
