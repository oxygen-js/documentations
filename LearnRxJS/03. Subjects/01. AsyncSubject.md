# AsyncSubject

## Испускает своё последнее значение при завершении



### Примеры

##### Пример 1: простой AsyncSubject

(
[Stackblitz](https://stackblitz.com/edit/rxjs-asyncsubject?file=index.ts&devtoolsheight=100)
)

```js
// RxJS v6+
import { AsyncSubject } from 'rxjs';

const sub = new AsyncSubject();

sub.subscribe(console.log);

sub.next(123); //ничего не логируется

sub.subscribe(console.log);

sub.next(456); //ничего не логируется
sub.complete(); //456, 456 логируются обоими подписчиками
```
