### Interceptors

Между [обычными interceptor'ами](<../02. Overview/09. Interceptors.md>) и interceptor'ами веб-сокетов нет никакой разницы. В примере ниже используется созданный вручную interceptor уровня метода. Как и в приложениях на основе HTTP, вы можете применять interceptor'ы уровня шлюза (то есть указать декоратор `@UseInterceptors()` перед классом шлюза).

```typescript
@UseInterceptors(new TransformInterceptor())
@SubscribeMessage('events')
handleEvent(client: Client, data: unknown): WsResponse<unknown> {
  const event = 'events';
  return { event, data };
}
```
