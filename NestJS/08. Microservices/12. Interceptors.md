### Interceptors

Между [обычными interceptor'ами](<../02. Overview/09. Interceptors.md>) и interceptor'ами микросервисов нет никакой разницы. В примере ниже используется вручную созданный interceptor уровня метода. Как и в приложениях на базе HTTP, можно применять interceptor'ы уровня контроллера (то есть указать декоратор `@UseInterceptors()` перед классом контроллера).

```typescript
@UseInterceptors(new TransformInterceptor())
@MessagePattern({ cmd: 'sum' })
accumulate(data: number[]): number {
  return (data || []).reduce((a, b) => a + b);
}
```
