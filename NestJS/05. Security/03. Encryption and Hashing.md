### Шифрование и хэширование

**Шифрование** — это процесс кодирования информации. В ходе него исходное представление информации, называемое открытым текстом, преобразуется в другую форму — шифротекст. В идеале расшифровать шифротекст обратно в открытый текст и получить доступ к исходной информации могут только уполномоченные стороны. Само по себе шифрование не предотвращает перехват, но лишает потенциального перехватчика возможности понять содержимое. Шифрование — двусторонняя функция: зашифрованное можно расшифровать при наличии подходящего ключа.

**Хэширование** — это процесс преобразования заданного ключа в другое значение. Новое значение вычисляется хэш-функцией по математическому алгоритму. После хэширования восстановить входные данные по результату должно быть невозможно.

#### Шифрование

Node.js предоставляет встроенный [модуль crypto](https://nodejs.org/api/crypto.html), с помощью которого можно шифровать и расшифровывать строки, числа, буферы, потоки и многое другое. Nest не добавляет поверх этого модуля никакого дополнительного пакета, чтобы не вводить лишних абстракций.

В качестве примера воспользуемся алгоритмом AES (Advanced Encryption System) `'aes-256-ctr'` в режиме шифрования CTR.

```typescript
import { createCipheriv, randomBytes, scrypt } from 'node:crypto';
import { promisify } from 'node:util';

const iv = randomBytes(16);
const password = 'Password used to generate key';

// Длина ключа зависит от алгоритма.
// В данном случае для aes256 это 32 байта.
const key = (await promisify(scrypt)(password, 'salt', 32)) as Buffer;
const cipher = createCipheriv('aes-256-ctr', key, iv);

const textToEncrypt = 'Nest';
const encryptedText = Buffer.concat([
  cipher.update(textToEncrypt),
  cipher.final(),
]);
```

Теперь расшифруем значение `encryptedText`:

```typescript
import { createDecipheriv } from 'node:crypto';

const decipher = createDecipheriv('aes-256-ctr', key, iv);
const decryptedText = Buffer.concat([
  decipher.update(encryptedText),
  decipher.final(),
]);
```

#### Хэширование

Для хэширования мы рекомендуем использовать пакет [bcrypt](https://www.npmjs.com/package/bcrypt) или [argon2](https://www.npmjs.com/package/argon2). Nest не предоставляет дополнительных обёрток поверх этих модулей, чтобы не вводить лишних абстракций (и не удлинять кривую обучения).

В качестве примера захэшируем случайный пароль с помощью `bcrypt`.

Сначала установите необходимые пакеты:

```shell
$ npm i bcrypt
$ npm i -D @types/bcrypt
```

После установки можно использовать функцию `hash`:

```typescript
import * as bcrypt from 'bcrypt';

const saltOrRounds = 10;
const password = 'random_password';
const hash = await bcrypt.hash(password, saltOrRounds);
```

Для генерации соли используйте функцию `genSalt`:

```typescript
const salt = await bcrypt.genSalt();
```

Для сравнения/проверки пароля используйте функцию `compare`:

```typescript
const isMatch = await bcrypt.compare(password, hash);
```

Подробнее о доступных функциях можно прочитать [здесь](https://www.npmjs.com/package/bcrypt).
