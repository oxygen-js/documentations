### Глобальный префикс

Чтобы задать префикс для **каждого маршрута**, зарегистрированного в HTTP-приложении, используйте метод `setGlobalPrefix()` экземпляра `INestApplication`.

```typescript
const app = await NestFactory.create(AppModule);
app.setGlobalPrefix('v1');
```

Исключить отдельные маршруты из глобального префикса можно следующей конструкцией:

```typescript
app.setGlobalPrefix('v1', {
  exclude: [{ path: 'health', method: RequestMethod.GET }],
});
```

Либо можно указать маршрут строкой (тогда он будет применён ко всем методам запроса):

```typescript
app.setGlobalPrefix('v1', { exclude: ['cats'] });
```

> **Подсказка** Свойство `path` поддерживает подстановочные параметры с помощью пакета [path-to-regexp](https://github.com/pillarjs/path-to-regexp#parameters). Обратите внимание: звёздочки `*` в качестве подстановочных символов не поддерживаются. Вместо них нужно использовать параметры (`:param`) или именованные подстановки (`*splat`).
