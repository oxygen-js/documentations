### HTTPS

Чтобы создать приложение, работающее по протоколу HTTPS, задайте свойство `httpsOptions` в объекте опций, передаваемом в метод `create()` класса `NestFactory`:

```typescript
const httpsOptions = {
  key: fs.readFileSync('./secrets/private-key.pem'),
  cert: fs.readFileSync('./secrets/public-certificate.pem'),
};
const app = await NestFactory.create(AppModule, {
  httpsOptions,
});
await app.listen(process.env.PORT ?? 3000);
```

Если вы используете `FastifyAdapter`, создавайте приложение так:

```typescript
const app = await NestFactory.create<NestFastifyApplication>(
  AppModule,
  new FastifyAdapter({ https: httpsOptions }),
);
```

#### Несколько серверов одновременно

Следующий рецепт показывает, как создать экземпляр приложения Nest, который одновременно слушает несколько портов (например, порт без HTTPS и порт HTTPS).

```typescript
const httpsOptions = {
  key: fs.readFileSync('./secrets/private-key.pem'),
  cert: fs.readFileSync('./secrets/public-certificate.pem'),
};

const server = express();
const app = await NestFactory.create(AppModule, new ExpressAdapter(server));
await app.init();

const httpServer = http.createServer(server).listen(3000);
const httpsServer = https.createServer(httpsOptions, server).listen(443);
```

Поскольку мы сами вызвали `http.createServer` / `https.createServer`, NestJS не закрывает эти серверы при вызове `app.close` или по сигналу завершения. Это нужно сделать самостоятельно:

```typescript
@Injectable()
export class ShutdownObserver implements OnApplicationShutdown {
  private httpServers: http.Server[] = [];

  public addHttpServer(server: http.Server): void {
    this.httpServers.push(server);
  }

  public async onApplicationShutdown(): Promise<void> {
    await Promise.all(
      this.httpServers.map(
        (server) =>
          new Promise((resolve, reject) => {
            server.close((error) => {
              if (error) {
                reject(error);
              } else {
                resolve(null);
              }
            });
          }),
      ),
    );
  }
}

const shutdownObserver = app.get(ShutdownObserver);
shutdownObserver.addHttpServer(httpServer);
shutdownObserver.addHttpServer(httpsServer);
```

> **Подсказка** `ExpressAdapter` импортируется из пакета `@nestjs/platform-express`. Пакеты `http` и `https` — встроенные пакеты Node.js.

> **Предупреждение** Этот рецепт не работает с [подписками GraphQL](<../06. GraphQL/04. Subscriptions.md>).
