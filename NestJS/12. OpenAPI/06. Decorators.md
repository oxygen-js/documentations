### Декораторы

Все доступные декораторы OpenAPI имеют префикс `Api`, чтобы отличать их от декораторов ядра. Ниже приведён полный список экспортируемых декораторов с указанием уровня, на котором декоратор может применяться.

|                           |                    |
| ------------------------- | ------------------ |
| `@ApiBasicAuth()`         | Метод / Контроллер |
| `@ApiBearerAuth()`        | Метод / Контроллер |
| `@ApiBody()`              | Метод              |
| `@ApiConsumes()`          | Метод / Контроллер |
| `@ApiCookieAuth()`        | Метод / Контроллер |
| `@ApiExcludeController()` | Контроллер         |
| `@ApiExcludeEndpoint()`   | Метод              |
| `@ApiExtension()`         | Метод              |
| `@ApiExtraModels()`       | Метод / Контроллер |
| `@ApiHeader()`            | Метод / Контроллер |
| `@ApiHideProperty()`      | Модель             |
| `@ApiOAuth2()`            | Метод / Контроллер |
| `@ApiOperation()`         | Метод              |
| `@ApiParam()`             | Метод / Контроллер |
| `@ApiProduces()`          | Метод / Контроллер |
| `@ApiSchema()`            | Модель             |
| `@ApiProperty()`          | Модель             |
| `@ApiPropertyOptional()`  | Модель             |
| `@ApiQuery()`             | Метод / Контроллер |
| `@ApiResponse()`          | Метод / Контроллер |
| `@ApiSecurity()`          | Метод / Контроллер |
| `@ApiTags()`              | Метод / Контроллер |
| `@ApiCallbacks()`         | Метод / Контроллер |
