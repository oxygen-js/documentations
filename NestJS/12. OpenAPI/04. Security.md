### Безопасность

Чтобы указать, какие механизмы безопасности должны применяться к конкретной операции, используйте декоратор `@ApiSecurity()`.

```typescript
@ApiSecurity('basic')
@Controller('cats')
export class CatsController {}
```

Перед запуском приложения не забудьте добавить определение схемы безопасности в базовый документ с помощью `DocumentBuilder`:

```typescript
const options = new DocumentBuilder().addSecurity('basic', {
  type: 'http',
  scheme: 'basic',
});
```

Некоторые из самых распространённых способов аутентификации встроены (например, `basic` и `bearer`), поэтому описывать механизмы безопасности вручную, как показано выше, не обязательно.

#### Basic-аутентификация

Чтобы включить basic-аутентификацию, используйте `@ApiBasicAuth()`.

```typescript
@ApiBasicAuth()
@Controller('cats')
export class CatsController {}
```

Перед запуском приложения не забудьте добавить определение схемы безопасности в базовый документ с помощью `DocumentBuilder`:

```typescript
const options = new DocumentBuilder().addBasicAuth();
```

#### Bearer-аутентификация

Чтобы включить bearer-аутентификацию, используйте `@ApiBearerAuth()`.

```typescript
@ApiBearerAuth()
@Controller('cats')
export class CatsController {}
```

Перед запуском приложения не забудьте добавить определение схемы безопасности в базовый документ с помощью `DocumentBuilder`:

```typescript
const options = new DocumentBuilder().addBearerAuth();
```

#### Аутентификация OAuth2

Чтобы включить OAuth2, используйте `@ApiOAuth2()`.

```typescript
@ApiOAuth2(['pets:write'])
@Controller('cats')
export class CatsController {}
```

Перед запуском приложения не забудьте добавить определение схемы безопасности в базовый документ с помощью `DocumentBuilder`:

```typescript
const options = new DocumentBuilder().addOAuth2();
```

#### Аутентификация через cookie

Чтобы включить аутентификацию через cookie, используйте `@ApiCookieAuth()`.

```typescript
@ApiCookieAuth()
@Controller('cats')
export class CatsController {}
```

Перед запуском приложения не забудьте добавить определение схемы безопасности в базовый документ с помощью `DocumentBuilder`:

```typescript
const options = new DocumentBuilder().addCookieAuth('optional-session-id');
```
