### Документация

**Compodoc** — инструмент генерации документации для приложений на Angular. Поскольку Nest и Angular имеют схожую структуру проектов и кода, **Compodoc** работает и с приложениями на Nest.

#### Настройка

Настроить Compodoc в существующем проекте Nest очень просто. Начните с добавления dev-зависимости — выполните в терминале вашей ОС следующую команду:

```bash
$ npm i -D @compodoc/compodoc
```

#### Генерация

Сгенерируйте документацию проекта следующей командой (для поддержки `npx` требуется npm 6). Другие опции описаны в [официальной документации](https://compodoc.app/guides/usage.html).

```bash
$ npx @compodoc/compodoc -p tsconfig.json -s
```

Откройте браузер и перейдите по адресу [http://localhost:8080](http://localhost:8080). Вы должны увидеть исходный проект Nest CLI:
