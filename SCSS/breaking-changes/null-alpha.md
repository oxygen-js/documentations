# Ломающее изменение: null в альфа-канале

До Dart Sass 1.64.3 в JS- и Dart-API передача `null` в конструктор `SassColor` трактовалась как значение 1. Теперь это поведение считается устаревшим. Вместо этого пользователям следует явно передавать 1 или `undefined`.

Sass работает над добавлением поддержки [CSS Color Module Level 4]. Одно из изменений в этом модуле — идея [«отсутствующих компонентов»]: если компонент цвета, например `alpha`, отсутствует, он в основном трактуется как 0, но при интерполяции с другим цветом (например, в градиенте или анимации) он автоматически принимает значение этого другого цвета.

[CSS Color Module Level 4]: https://www.w3.org/TR/css-color-4/
[«отсутствующих компонентов»]: https://developer.mozilla.org/en-US/docs/Web/CSS/color_value#missing_color_components

Пользователям JS- и Dart-API нужен способ получать и задавать отсутствующие каналы, и `null` — самый естественный способ сделать это. В большинстве случаев это не проблема: те, кто хочет создать непрозрачный цвет, обычно просто не указывают параметр `alpha` (или передают `undefined` в JS). Но если вызывающий код явно передаёт `null`, со временем это будет трактоваться как прозрачный цвет, а не как непрозрачный.

Чтобы сохранить текущее поведение, достаточно явно передавать 1, если `alpha` не задан. В JS:

```js
new sass.SassColor({
  red: 102,
  green: 51,
  blue: 153,
  alpha: alpha ?? 1,
});
```

А в Dart:

```dart
sass.SassColor.rgb(102, 51, 153, alpha ?? 1);
```

> 💡 **Занятный факт**
>
> Типы TypeScript для Sass API уже запрещают передавать `null` в качестве `alpha` — допускается только отсутствие значения, `undefined` или `Number`. Но до Dart Sass 1.64.3, если вы не использовали TypeScript и всё же передавали `null`, он всё равно трактовался как непрозрачный цвет.

## Переходный период

**Совместимость:** Dart Sass: начиная с 1.64.3 · LibSass: ✗ · Ruby Sass: ✗


Между Dart Sass 1.64.3 и будущим релизом с поддержкой CSS Colors Level 4 Dart Sass продолжит трактовать значение `alpha`, равное `null`, как непрозрачный цвет. Однако он будет выводить предупреждение об устаревании, чтобы побудить авторов явно передавать `alpha` равным 1.
